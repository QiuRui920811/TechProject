## 一階 — 工業基礎

**解鎖成本**: 物品 14 經驗 / 機器 30 經驗

初始免費解鎖的機器：碎礦機、自動農場、電鋸、回收機、太陽能發電機、燃煤發電機

### 核心產線

```
鐵礦 ──[碎礦機]──→ 鐵粉 ──[電爐]──→ 鐵錠 ──[壓縮器]──→ 鐵板
                                                         ↓
銅礦 ──[碎礦機]──→ 銅粉 ──[電爐]──→ 銅錠 ──[製造機]──→ 電路板
                                                         ↓
石英 ──[碎礦機]──→ 矽  ───────────────────[製造機]──→ 機械零件
                                                         ↓
原木 ──[電鋸] ──→ 木屑                               精密框架
```

---

## 二階 — 中期擴展

**解鎖成本**: 物品 32 經驗 / 機器 64 經驗

### 精煉鏈

```
鐵粉 ──[洗礦機]──→ 礦漿 ──[淨化器]──→ 淨化碎晶 ──[離心機]──→ 離子粉
```

### 生質鏈

```
作物+木屑 ──[生質實驗室]──→ 生質樹脂 ──[化學反應器]──→ 聚合樹脂
                                       ↓
                              ──[聚合壓床]──→ 橡膠片
```

---

## 三階 — 高階科技

**解鎖成本**: 物品 72 經驗 / 機器 128 經驗

### 量子鏈

```
進階電路 + 奈米塗層 ──[量子處理器]──→ 量子晶片
磁環 + 能源電池 ──[量子處理器]──→ 通量連結 → 量子流體
                                              ↓
紫水晶+冷卻液 ──[晶體培育艙]──→ 晶種 → 培育晶體
                                              ↓
              ──[力場鍛造台]──→ 力場板
                    ↓
              ──[聚變反應爐]──→ 聚變核心
                    ↓
              ──[物質編譯器]──→ 奇點碎片
```

---

## 四階 — 終局

**解鎖成本**: 物品 132 經驗 / 機器 208 經驗

13 台機器形成嚴格的線性鏈條，從虛空抽取器開始，到歐米茄製造器結束。

詳見 [四階機器](machines/tier-4.md) 和 [終局材料](items/endgame-materials.md)。

## 巨構 — 科技巔峰

8 台巨構機器，能耗 極高，製造宇宙最高端的材料。

最終產物：**歐米茄檔案**

詳見 [巨構機器](machines/megastructures.md)。

## 平行路線

### 擴展科技線 (二～三階)

```
窯爐 → 絕緣壓機 → 低溫蒸餾器 (熱/絕緣/低溫)
基因剪接台 → 生物合成槽 (生物合成)
電漿精煉器 → 光子編織機 (電漿/光子)
天文觀測台 → 測繪信標 → 遺物解析儀 (天文/行星)
異星精煉台 → 行星鍛造台 (行星裝備)
```

### 行星路線 (三階)

```
天文觀測台 → 測繪信標 → 星門 → 行星探索
              ↓
        遺物解析儀 → 異星精煉台 → 行星鍛造台
              ↓              ↓
        古代訊號      異星合金 → 防護套裝
```

### 系統自動化路線 (二～三階)

```
礦場鑽機 Mk1/Mk2/Mk3 (自動採礦)
作物收割器 / 伐木機 / 釣魚碼頭 (資源收集)
儲存中心 / 過濾路由器 / 貨物系統 (物流管理)
安卓機器人 (全自動化)
```
