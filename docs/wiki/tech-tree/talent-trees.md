# 天賦樹系統

TechProject 擁有 6 棵天賦樹，每棵包含 15 個節點。投資天賦點可以提升角色屬性，並解鎖強力的主動法術。

---

## 系統概覽

| 項目 | 說明 |
|---|---|
| 天賦樹數量 | 6 棵 |
| 每棵節點數 | 15 |
| 最高等級 | 50（每升一級獲得 1 天賦點） |
| 重置費用 | 1,000 研究 XP（退還所有已投資點數） |
| 里程碑 | Lv.5 / 10 / 15 / 20 / 25 / 30 / 40 / 50 各有獎勵 |

---

## 節點類型

| 類型 | 每等費用 | 最大等級 | 說明 |
|---|---|---|---|
| 起始 (START) | 0 | 1 | 免費入口點 |
| 次要 (MINOR) | 1 點 | 5 | 每級 +小幅屬性加成 |
| 重要 (NOTABLE) | 2 點 | 1 | 一次性中幅加成 |
| 基石 (KEYSTONE) | 3 點 | 1 | 解鎖主動法術 |

---

## 屬性系統

天賦加成會疊加到技能等級提供的基礎屬性上。共 13 項屬性：

| 屬性 | 基礎值 | 主要來源技能 |
|---|---|---|
| 最大生命 | 20 | 工程 / 採集 |
| 最大魔力 | 100 | 研究 / 共鳴 |
| 魔力回復 | 4.0/秒 | 共鳴 / 研究 |
| 生命回復 | 0.5/秒 | 採集 / 工程 |
| 力量 | 1.0 | 戰鬥 |
| 防禦 | 0.0 | 工程 / 戰鬥 |
| 速度 | 0.0 | 探索 |
| 暴擊率 | 5% | 探索 / 研究 |
| 暴擊傷害 | 50% | 戰鬥 / 共鳴 |
| 效率 | 0.0 | 工程 / 採集 |
| 幸運 | 0.0 | 採集 |
| 魔法強度 | 0.0 | 研究 / 共鳴 |
| 治療強度 | 0.0 | 共鳴 / 採集 |
