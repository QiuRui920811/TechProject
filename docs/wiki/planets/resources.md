# 星球資源用途總覽

每顆星球除了挑戰本身，**獨有資源都有對應的加工/強化配方**——讓打星球的回報感最大化。

> 🆕 2026/04/29 更新：新增 4 條作物加工線 + 3 條 Relic 強化配方，原本「採回來沒用」的星球作物現在全部有去處。

---

## 🌿 星球作物加工（A 系列）

每顆星球的特色作物都有專屬轉換配方，把作物變成科技材料：

| 星球 | 作物 | 配方 | 機器 | 產出 |
|---|---|---|---|---|
| 🔥 **赫利昂** | 焰綻簇 ×4 + 紅石 | 高溫氧化 | 化學反應器 | 紅石 ×3 |
| 🔥 **赫利昂** | 焰根束 ×4 | 炭化 | 粉碎機 | 碳粉 ×2 |
| ❄️ **克里昂** | 霜綻花 ×4 + 水凝膠 | 低溫冷凝 | 低溫蒸餾器 | 冷卻電池 |
| 🌌 **倪克斯** | 回響孢子 ×4 + 邏輯閘 + 銅線 ×2 | 聲波感應 | 製造機 | 訊號中繼器 |

**為什麼這樣設計**：每個作物的「主題」對應星球氛圍（火 → 紅石/碳；冰 → 冷卻；聲波 → 信號），不是憑空產出，而是讓玩家在該星球大量採集作物時有意義。

## ⭐ Relic 強化配方（B 系列）

每顆中後期星球的 **Relic（遺物）** 除了原本的科技書解析路徑，現在還能**直接增強終局材料配方**——產量翻倍：

### 鈽燃料棒（赫利昂 Relic 強化）

| 配方 | 輸入 | 輸出 |
|---|---|---|
| 標準（原版） | 鈽 ×4 + 鉛 ×2 + 鋼棒 | 鈽燃料棒 ×1 |
| **赫利昂強化** | 鈽 ×3 + 鉛 ×2 + 鋼棒 + **赫利昂遺物 ×1** | **鈽燃料棒 ×2** |

→ 用 1 個 helion_relic + 少 1 顆鈽 換 +1 個鈽燃料棒。

### 強化合金錠（坦佩斯特 Relic 強化）

| 配方 | 輸入 | 輸出 |
|---|---|---|
| 標準 | 硬化合金 + 合成鑽石 + 濃縮鈾 | 強化合金錠 ×1 |
| **坦佩斯特強化** | 硬化合金 + 合成鑽石 + 濃縮鈾 + **坦佩斯特遺物 ×1** | **強化合金錠 ×2**（雷擊強化）|

### 聚變核心（倪克斯 Relic 強化）

| 配方 | 輸入 | 輸出 |
|---|---|---|
| 標準 | 聚變網 ×2 + 穩定核心 | 聚變核心 ×1 |
| **倪克斯強化** | 聚變網 ×2 + 穩定核心 + **倪克斯遺物 ×1** | **聚變核心 ×2**（量子穩定）|

## 🏆 為什麼要打星球（投報整理）

| 星球 | 你能換到什麼 |
|---|---|
| **奧蕾莉亞**（1階） | 入門資源、低階遺物、新手探索 |
| **克里昂**（2階） | `cryon_relic` → 古代訊號 ／ 霜綻花 → 冷卻電池 |
| **倪克斯**（3階） | `nyx_relic` → 古代訊號 ／ 回響孢子 → 訊號中繼器 ／ **聚變核心 ×2** |
| **赫利昂**（4階） | `helion_relic` → 前線核心 ／ 焰綻簇 → 紅石 ／ 焰根束 → 碳粉 ／ **鈽燃料棒 ×2** |
| **坦佩斯特**（5階） | `tempest_relic` → 前線核心 ／ **強化合金錠 ×2** |
| **迷途星**（6階） | `labyrinth_relic` → 完整迷宮裝備系列（劍/弩/鎬/盔甲/羅盤）|

---

## 💡 玩法建議

- **早期**：打到 cryon/nyx 後，作物先別亂丟，回基地走加工配方換實用材料
- **中期**：拿到 helion_relic 就先存著，**等你做鈽燃料棒時用強化版**（產量直接翻倍）
- **後期**：tempest_relic + nyx_relic 的強化配方對 endgame 量產極有幫助
- **長期**：迷途星的回報是裝備系列，跟 relic 強化配方互補
