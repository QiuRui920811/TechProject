# 星門建造

行星星門是連接地球與外星球的傳送裝置。

---

## 星門屬性

| 屬性 | 值 |
|---|---|
| 機器名稱 | 行星星門 |
| 階級 | TIER 3 |
| 啟動能耗 | **1200 EU** |
| 閒置消耗 | 12 EU/tick |

---

## 星門建造

### 所需材料

在進階製造機中合成：

| 材料 | 來源 | 能耗 |
|---|---|---|
| 重力軸承 | 天文觀測台 (磁環 + 重力子透鏡) | 24 EU |
| 量子框架 | 量子處理器 (通量連結 + 強化玻璃) | 22 EU |
| 異星合金 | 異星精煉台 (古代訊號 + 輻照碎片) | 28 EU |

最終合成：**重力軸承 + 量子框架 + 異星合金** → 行星星門 (34 EU)

### 前置科技鏈

```
天文觀測台 → 星圖 → 測繪晶片
     ↓
測繪信標 → 測繪資料
     ↓
探索初階星球 → 行星遺物 → 遺物解析儀 → 古代訊號
                                          ↓
                                   異星精煉台 → 異星合金
                                          ↓
                               進階製造機 → 行星星門
```

---

## 星門使用

### 步驟

1. 放置星門方塊
2. 確保能源網路供應至少 **1200 EU**
3. 右鍵點擊星門開啟選單
4. 選擇目的地星球
5. 確認傳送

### 星門選單

| 位置 | 目的地 |
|---|---|
| 🌍 | 地球 (返回) |
| ☢️ | 奧蕾莉亞 |
| ☀️ | 赫利昂 |
| 🌑 | 倪克斯 |
| ❄️ | 克里昂 |
| ⚡ | 坦佩斯特 |

---

## 能源需求

星門是最耗能的設施之一：

- 每次傳送消耗 **1200 EU**
- 閒置時仍持續消耗 12 EU/tick
- 建議搭配太陽能陣列或聚變反應爐供電
- 可使用電池庫儲存能量

### 推薦供電方案

```
聚變反應爐 → 能源節點 → 電池庫 → 星門
               或
太陽能陣列 (多台) → 能源節點 → 電池庫 → 星門
```

---

## 注意事項

- 傳送後會出現在星球的隨機位置
- 行星邊界半徑 3000 方塊，超出會被傳回
- 確保穿著對應防護套裝再傳送
- 可在星球建立多個星門方便返回
