## 星門旅行

詳見 [星門建造](stargate.md)

- 星門能耗: **1200 EU** (啟動)
- 閒置消耗: **12 EU/tick**
- 邊界半徑: **3000 方塊**

### 旅行流程

```
建造星門 → 供能 1200 EU → 選擇目的地 → 傳送
    ↓
 抵達星球 → 穿著防護套裝 → 探索/採集 → 返回 (地球)
```

## 星球分級

### 初階星球 (TIER 3 早期)

**奧蕾莉亞、克里昂、倪克斯**

- 使用 `測繪資料` 啟動遺跡
- 各自對應單一防護套裝
- 獲得各星球獨有遺物

### 進階星球 (TIER 3 後期)

**赫利昂、坦佩斯特**

- 使用 `古代訊號` 啟動遺跡
- 需要**前線套裝** (可抵禦所有危害)
- 獲得前線核心碎片
