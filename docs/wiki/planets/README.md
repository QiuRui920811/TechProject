## 星球分級

### 初階星球 (三階早期)

**奧蕾莉亞、克里昂、倪克斯**

- 使用 `測繪資料` 啟動遺跡
- 各自對應單一防護套裝
- 獲得各星球獨有遺物

### 進階星球 (三階後期)

**赫利昂、坦佩斯特**

- 使用 `古代訊號` 啟動遺跡
- 需要**前線套裝** (可抵禦所有危害)
- 獲得前線核心碎片

## 遺跡挑戰

探索星球時可找到遺跡核心方塊，手持啟動物品右鍵啟動後進入波狀戰鬥挑戰。

### 挑戰機制

- **共 3 波**，每波 90 秒獨立倒數計時
- 波間有 4 秒準備間隔，不計入倒數
- 第 1 波 3 隻、第 2 波 5 隻、第 3 波 7 隻
- 後續波次怪物更強 (血量/攻擊力遞增)
- 第 3 波怪物具有精英強化效果
- 任何一波超時或玩家陣亡 → 挑戰失敗，核心休眠（冷卻 30 分鐘）
- 全部清完 → 獲得行星遺物等獎勵
- 波次推進由 `AtomicBoolean` 守衛，避免競態條件造成跳波（歷史 bug 已修復）

### Boss Bar 倒數

挑戰期間畫面上方顯示 BossBar，包含：
- 當前波次 / 總波數
- 剩餘敵人數量
- 剩餘秒數（以進度條呈現）
