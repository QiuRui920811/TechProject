## 輻射防護

### 抗輻射套裝

在行星鍛造台中合成：

| 部件 | 材料 |
|---|---|
| 頭盔 | 外骨骼框架 + 密封織層 + 輻射網層 |
| 胸甲 | 外骨骼框架 + 密封織層 + 輻射網層 |
| 護腿 | 外骨骼框架 + 密封織層 + 輻射網層 |
| 戰靴 | 外骨骼框架 + 密封織層 + 輻射網層 |

**輻射網層** = 異星合金 + 輻照碎片 + 寄生腺體

### 反食物

**輻晶蜜凍** — 微光果 + 輻照碎片 (生質實驗室 —)

## 探索建議

- 首次登陸前在生質實驗室合成「**輻射濾除劑**」（生質催化劑 + 神經樹脂），提供 90 秒輻射防護
- 多帶幾顆輻射濾除劑，到星球後採集輻照碎片與擊殺菁英取得寄生腺體
- 回地球後做出完整抗輻射套裝，再二次登陸深入探索
- 收集虛空花簮種子帶回地球種植（農地即可種下）
