## 遺跡探索

1. 探索星球找到遺跡（重生錨結構）
2. 手持**測繪資料**右鍵點擊重生錨
3. 獲得**行星遺物**和**古代訊號**

## 輻射防護

### 抗輻射套裝

在行星鍛造台中合成：

| 部件 | 材料 |
|---|---|
| 頭盔 | 外骨骼框架 + 密封織層 + 輻射網層 |
| 胸甲 | 外骨骼框架 + 密封織層 + 輻射網層 |
| 護腿 | 外骨骼框架 + 密封織層 + 輻射網層 |
| 戰靴 | 外骨骼框架 + 密封織層 + 輻射網層 |

**輻射網層** = 異星合金 + 輻照碎片 + 寄生腺體

### 反食物

**輻晶蜜凍** — 微光果 + 輻照碎片 (生質實驗室 18 EU)

---

## 探索建議

- 先建造抗輻射套裝再登陸
- 攜帶輻晶蜜凍作為緊急食物
- 多收集輻照碎片和虛空花種子帶回地球
- 擊殺特殊怪物取得寄生腺體
