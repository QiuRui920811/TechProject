1. 探索星球找到遺跡（重生錨結構）
2. 手持**測繪資料**右鍵點擊重生錨
3. 獲得**行星遺物**和**古代訊號**

## 輻射防護

### 抗輻射套裝

在行星鍛造台中合成：

| 部件 | 材料 |
|---|---|
| 頭盔 | 外骨骼框架 + 密封織層 + 輻射網層 |
| 胸甲 | 外骨骼框架 + 密封織層 + 輻射網層 |
| 護腿 | 外骨骼框架 + 密封織層 + 輻射網層 |
| 戰靴 | 外骨骼框架 + 密封織層 + 輻射網層 |

**輻射網層** = 異星合金 + 輻照碎片 + 寄生腺體

### 反食物

**輻晶蜜凍** — 微光果 + 輻照碎片 (生質實驗室 —)

## 星球怪物

### 怪物強化

奧蕾莉亞的一般怪物會被強化：

| 屬性 | 強化值 |
|---|---|
| 生命值 | × 1.25 |
| 攻擊力 | + 1.0 |
| 移動速度 | 略微提升 |

### 怪物技能

一般怪物攻擊玩家時有 **30%** 機率觸發技能（冷卻 6 秒）：
- **輻射感染**：施加中毒效果（50 ticks）

### 輻塵寄生體（菁英）

| 屬性 | 值 |
|---|---|
| 血量倍率 | × 2.2 |
| 攻擊加成 | + 3.0 |
| 移動速度 | + 0.025 |

- 掉落 **寄生腺體** (`aurelia_parasite_gland`)
- 額外掉落 **輻照碎片** (`irradiated_shard`)

## 探索建議

- 首次登陸前在生質實驗室合成「**輻射濾除劑**」（生質催化劑 + 神經樹脂），提供 90 秒輻射防護
- 多帶幾顆輻射濾除劑，到星球後採集輻照碎片與擊殺菁英取得寄生腺體
- 回地球後做出完整抗輻射套裝，再二次登陸深入探索
- 收集虛空花簮種子帶回地球種植（農地即可種下）
