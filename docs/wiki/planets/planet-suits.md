## 前線套裝

**對抗**：所有五種環境危害

前線套裝是最終極的防護裝備，可以在任何星球自由活動。

### 前線核心

合成前線套裝的核心材料：

```
前線核心碎片 (遺跡獲得)
    + 異星合金
    + 灰燼燃核 (赫利昂怪物)
    + 電容核 (坦佩斯特怪物)
    ↓
前線核心 (行星鍛造台 34 EU)
```

### 專用材料

- **耀焰屏護層** = 異星合金 + 日灼碎片 + 灰燼燃核 (31 EU)
- **風暴導流層** = 異星合金 + 暴嵐玻片 + 電容核 (31 EU)

| 部件 | 合成 | 能耗 |
|---|---|---|
| 頭盔 | 前線核心 + 耀焰屏護層 + 風暴導流層 + 玻璃板 | 36 EU |
| 胸甲 | 前線核心 + 耀焰屏護層 + 風暴導流層 + 碳纖網 | 38 EU |
| 護腿 | 前線核心 + 耀焰屏護層 + 風暴導流層 + 絕緣線材 | 37 EU |
| 戰靴 | 前線核心 + 耀焰屏護層 + 風暴導流層 + 橡膠片 | 35 EU |

## 合成路線建議

```
初階三星球探索 → 遺物 + 怪物掉落
    ↓
遺物解析儀 → 古代訊號 → 異星精煉台 → 異星合金
    ↓
行星鍛造台 → 初階套裝 (一次做一套)
    ↓
進階星球 (赫利昂 + 坦佩斯特) → 灰燼燃核 + 電容核
    ↓
前線套裝 (最終目標)
```
