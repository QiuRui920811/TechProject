## 進階星球

迷途星是**進階星球**，與赫利昂、坦佩斯特同屬三階後期目標：

- 需要 `古代訊號` 啟動遺跡
- 需要**前線套裝**抵禦瘴氣危害
- 掉落 **迷宮遺物** 與 **前線核心碎片**

## 世界結構

### The Glade —— 倖存者基地 (中心 100×100)

世界座標 `±50 x ±50` 範圍內是唯一的安全區域：

- **三格厚石牆**完全封閉，四方各一座鍍金黑石門框（N / S / E / W）
- **中央篝火廣場**：5×5 石磚平台 + 中心篝火 + 四角燈籠
- 門前通道鋪設**黑石磚道路**通向四個大門
- 門框頂部擺放**靈魂燈籠**作為地標
- 基地內有小木屋等結構，可作為短暫休憩點
- 門由 MazeService 動態管理；玩家可從門洞進出迷宮

### 外部迷宮 —— 無盡 9×9 格迷宮

Glade 外是一座**無盡延伸的生成式迷宮**（spanning tree 演算法）：

- **格子尺寸**：9×9，**牆厚 3 格**，**牆高 100 格**
- 生成範圍約 ±490 格（108 cells），但實際由**世界邊界 2000×2000** 限制探索範圍
- 約 **15%** 額外隨機通道，避免死路過多，維持探索路線多樣性
- **交叉角柱**使用 `CHISELED_DEEPSLATE` 作頂蓋
- 牆壁材質多重噪點：`MOSS_BLOCK` / `SCULK` / `BONE_BLOCK` / `SOUL_SOIL` / `CRACKED_DEEPSLATE_BRICKS` / `COBBLED_DEEPSLATE` / `DEEPSLATE_TILES` / `CRACKED_DEEPSLATE_TILES` / `DEEPSLATE_BRICKS`
- 牆腳 3 層為苔蘚 + 深板岩斑駁過渡
- 中段稀疏放置**發光苔蘚**，避免完全黑暗但仍保留危險感

## 生物群系與氛圍

- **生物群系**：`DARK_FOREST`（黑森林），支援原版怪物自然生成
- **時間**：永久午夜（天空時間固定為 18000）
- **怪物生成上限**：120（標準 70 的 ~1.7 倍）
- **環境生成上限**：40
- **原版怪物**（殭屍、骷髏、蜘蛛、苦力怕、末影人等）會在黑森林中自然生成
- Tech 插件自動關閉迷途星的裝飾 / 結構 / 洞穴生成，但**保留怪物生成**以營造詭譎氛圍

## 遺跡挑戰

迷途星遺跡核心為 **監守尖嘯器 (Sculk Shrieker)**，啟動物品為**古代訊號**。

### 挑戰流程

1. 探索迷宮找到監守尖嘯器遺跡
2. 手持**古代訊號**右鍵啟動
3. 進入 3 波戰鬥挑戰（與其他星球共用機制）

### 迷途星專屬怪物池

挑戰波怪物為**地面型組合**（不含飛行單位，避免越牆卡住）：

| 權重 | 單位 |
|---|---|
| 40% | 殭屍 (ZOMBIE) |
| 30% | 洞穴蜘蛛 (CAVE_SPIDER) |
| 20% | 骷髏 (SKELETON) |
| 10% | 蠹魚 (SILVERFISH) |

> 💡 **機制修正**：過去波次推進存在競態條件，偶爾會出現「跳波」現象。現已改用 `AtomicBoolean` 守衛，確保每波僅觸發一次推進。

### 挑戰獎勵

- 迷宮遺物 × 數個
- 前線核心碎片
- 挑戰結束後核心進入 30 分鐘冷卻

## 攻略要點

1. **首先鞏固 The Glade**：出星門後第一件事是確認自己站在倖存者基地內，記住四個大門對應的方向，避免迷路。
2. **沿牆探索**：迷宮使用 spanning tree，沿單一側牆走（左手或右手法則）雖然不保證最短路徑，但可避免鬼打牆。
3. **前線套裝必備**：瘴氣會持續消耗生命值，務必在前往前完成前線套裝。
4. **帶備信標**：建議攜帶光源與建築方塊搭建臨時避難點；迷宮牆高達 100 格，無法輕易翻牆。
5. **SCULK 節點採集**：使用**迷宮鎬**可顯著提升深岩類節點採集效率。
6. **菁英守衛者**：遇到菁英化的迷宮守衛者時建議優先擊殺，守衛者核心是迷宮刃合成的關鍵材料。
7. **注意世界邊界**：邊界僅 2000×2000，越界有 2 點/秒傷害，但實際上邊界內已有足夠探索空間。
