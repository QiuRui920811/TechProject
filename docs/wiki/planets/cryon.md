## 遺跡探索

1. 探索星球找到遺跡（磁石結構）
2. 手持**測繪資料**右鍵點擊磁石
3. 獲得**克里昂遺物**和**古代訊號**

## 低溫防護

### 低溫套裝

在行星鍛造台中合成：

| 部件 | 材料 |
|---|---|
| 頭盔 | 外骨骼框架 + 密封織層 + 低溫內襯 |
| 胸甲 | 外骨骼框架 + 密封織層 + 低溫內襯 |
| 護腿 | 外骨骼框架 + 密封織層 + 低溫內襯 |
| 戰靴 | 外骨骼框架 + 密封織層 + 低溫內襯 |

**低溫內襯** = 異星合金 + 低溫晶體 + 冰脈核心

### 反食物

**霜芯暖鍋** — 霜蘋果 + 低溫晶體 (生物合成槽 —)

## 星球怪物

### 怪物強化

克里昂的一般怪物會被強化：

| 屬性 | 強化值 |
|---|---|
| 生命值 | × 1.35 |
| 攻擊力 | + 1.5 |
| 移動速度 | 略微提升 |

### 怪物技能

一般怪物攻擊玩家時有 **30%** 機率觸發技能（冷卻 6 秒）：
- **極寒凍結**：施加緩速效果 + 凍結玩家

### 霜脊潛獵者（菁英）

| 屬性 | 值 |
|---|---|
| 血量倍率 | × 2.4 |
| 攻擊加成 | + 2.5 |
| 移動速度 | + 0.02 |

- 掉落 **冰脈核心** (`cryon_ice_heart`)
- 額外掉落 **低溫晶體** (`cryonite_crystal`)

## 探索建議

- 穿著低溫套裝防止凍傷
- 攜帶霜芯暖鍋應對突發狀況
- 收集低溫晶體和霜花種子
- 擊敗冰原怪物取得冰脈核心
