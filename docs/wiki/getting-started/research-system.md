# 研究系統

研究系統是 TechProject 的核心進度機制。透過累積研究 XP，你可以逐步解鎖新的物品與機器。

---

## 研究 XP

### 獲取方式

| 來源 | 說明 |
|---|---|
| 機器加工 | 每次機器完成一道配方即獲得 XP |
| 成就獎勵 | 完成成就可獲得大量 XP |
| 星球探索 | 啟動遺跡、收集樣本等 |
| 科技互動 | 各種科技系統的互動行為 |

### 查看 XP

- 使用 `/tech stats` 查看當前 XP 與等級
- PlaceholderAPI 佔位符: `%techproject_xp%`、`%techproject_level%`

---

## 解鎖機制

### 解鎖費用

| 階級 | 解鎖物品 | 解鎖機器 | 互動費用 |
|---|---|---|---|
| TIER 1 | 14 XP | 30 XP | 20 XP |
| TIER 2 | 32 XP | 64 XP | 44 XP |
| TIER 3 | 72 XP | 128 XP | 92 XP |
| TIER 4 | 132 XP | 208 XP | 152 XP |

### 解鎖前提

大多數物品和機器都有「前置解鎖」要求。例如：

- **進階電路** 需要先解鎖 **進階製造機**
- **量子晶片** 需要先解鎖 **量子處理器**
- **力場板** 需要先解鎖 **力場鍛造台**

解鎖鏈會自然引導你沿著科技樹前進。

---

## 初始解鎖

以下項目在遊戲開始時即可直接解鎖，無需前置：

- 碎礦機 (Crusher)
- 自動農場 (Auto Farm)
- 電鋸 (Electric Saw)
- 回收機 (Recycler)
- 太陽能發電機 (Solar Generator)
- 燃煤發電機 (Coal Generator)

---

## 科技百科

使用 `/tech book` 指令可以打開科技百科 GUI，其中包含：

- 所有物品的詳細說明
- 配方查詢
- 科技樹導覽
- 35+ 個教學章節

你也可以在物品欄中使用「科技百科」物品來開啟。
