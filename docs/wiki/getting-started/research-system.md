# 研究系統

透過累積研究經驗值，你可以逐步解鎖新的物品與機器。

---

## 如何獲得研究經驗

| 來源 | 說明 |
|---|---|
| 機器加工 | 每次機器完成一道配方即獲得經驗 |
| 成就獎勵 | 完成成就可獲得大量經驗 |
| 星球探索 | 啟動遺跡、收集樣本等 |
| 科技互動 | 各種科技系統的互動行為 |

使用 `/tech stats` 即可查看你目前的研究經驗與等級。

---

## 解鎖機制

大多數物品和機器都有「前置解鎖」要求，例如：

- 要做 **進階電路**，需要先解鎖 **進階製造機**
- 要做 **量子晶片**，需要先解鎖 **量子處理器**
- 要做 **力場板**，需要先解鎖 **力場鍛造台**

越高階的物品需要越多研究經驗才能解鎖。解鎖鏈會自然引導你沿著科技樹前進。

---

## 初始可解鎖

以下機器在遊戲開始時就可以直接解鎖，不需要任何前置：

- 碎礦機
- 自動農場
- 電鋸
- 回收機
- 太陽能發電機
- 燃煤發電機

---

## 科技百科

使用 `/tech book` 可以打開遊戲內的科技百科，裡面有：

- 所有物品的詳細說明
- 配方查詢
- 科技樹導覽

你也可以直接右鍵手上的「科技百科」物品來開啟。
