# 能源基礎

所有機器都需要能源 (EU) 才能運作。本頁介紹發電機與能源傳輸的基礎知識。

---

## 發電機

### 太陽能發電機

| 屬性 | 值 |
|---|---|
| 階級 | TIER 1 |
| 方塊 | 日照感測器 |
| 發電條件 | 白天且無遮擋 |
| 能耗 | 0 EU/tick（免費） |

**合成配方**:
```
玻璃    玻璃    玻璃
鐵錠    銅錠    鐵錠
鐵錠    紅石    鐵錠
```

> 💡 太陽能發電機是最推薦的初期發電方式，放在開闊的地方即可免費發電。

### 燃煤發電機

| 屬性 | 值 |
|---|---|
| 階級 | TIER 1 |
| 方塊 | 熔爐 |
| 燃料 | 煤炭、木炭等可燃物 |

**合成配方**:
```
鐵錠    鐵錠    鐵錠
鐵錠    熔爐    鐵錠
鐵錠    紅石    鐵錠
```

> 💡 燃煤發電機在夜間或地下基地中是太陽能的好搭檔。

## 進階發電

| 發電設備 | 階級 | 說明 |
|---|---|---|
| 太陽能陣列 | TIER 2 | 進階太陽能，輸出更高 |
| 風暴渦輪 | TIER 2 | 利用風力發電 |
| 聚變反應爐 | TIER 3 | 高產能聚變發電 |

## 能源傳輸

### 能源線纜

- **導能線纜** (Energy Cable): 基礎能源傳輸線
- 合成: 3 × 銅線 (水平排列)
- 放置在發電機與機器之間以傳輸電力

### 能源節點

- **能源節點** (Energy Node): 能源網路的中繼站
- 合成: 4 × 鐵錠 + 1 × 發射節點 (十字排列)
- 可以將能源分配給多台機器

---

## 能源消耗

每台機器都有 `energy-per-tick` 值，表示每個機器 tick 消耗的 EU：

| 機器類型 | 典型能耗 |
|---|---|
| TIER 1 機器 | 4-12 EU/tick |
| TIER 2 機器 | 9-18 EU/tick |
| TIER 3 機器 | 18-28 EU/tick |
| TIER 4 機器 | 28-52 EU/tick |

> ⚡ 機器 tick 週期預設為 20 ticks（1 秒），可在設定檔中調整。

---

## 電池庫

電池庫 (Battery Bank) 可以存儲能源，在發電不足時釋放：

**合成配方**:
```
鐵錠    鋰電池    鐵錠
鐵錠    能源電池  鐵錠
鐵錠    發射節點  鐵錠
```

> 💡 建議在太陽能發電機旁搭配電池庫，白天儲能、夜間釋放。
