# 能源基礎

所有機器都需要能源才能運作。本頁介紹發電機與能源傳輸。

## 能源傳輸

### 導能線纜

把發電機與機器連接起來的電線。用 3 個銅線水平排列合成。

### 能源節點

能源網路的中繼站，可以把電力分配給多台機器。

---

## 能源消耗參考

| 機器類型 | 典型能耗 |
|---|---|
| TIER 1 機器 | 4-12 EU/tick |
| TIER 2 機器 | 9-18 EU/tick |
| TIER 3 機器 | 18-28 EU/tick |
| TIER 4 機器 | 28-52 EU/tick |

---

## 電池庫

電池庫可以存儲能源，在發電不足時自動釋放。建議搭配太陽能發電機，白天儲能、夜間釋放。
