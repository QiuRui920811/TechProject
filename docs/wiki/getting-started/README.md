# 🚀 新手入門

歡迎來到 TechProject 科技世界！本章節將帶你從零開始，建造第一台機器並踏上科技之路。

---

## 開始之前

TechProject 是一個漸進式科技插件。你需要：

1. **收集原版材料** — 鐵錠、銅錠、紅石、鑽石等
2. **建造研究台** — 這是解鎖所有科技的核心
3. **研究物品/機器** — 花費研究 XP 來解鎖新科技
4. **建造機器** — 在工作台中合成機器方塊
5. **供給能源** — 機器需要電力才能運作

---

## 核心流程

```
收集原版材料
    ↓
建造 研究台 (Research Desk)
    ↓
研究並解鎖 碎礦機 + 發電機
    ↓
建造 太陽能發電機 或 燃煤發電機
    ↓
建造 碎礦機，開始雙倍產礦
    ↓
逐步解鎖更多機器與材料...
```

---

以下機器在遊戲開始時即可解鎖（初始解鎖物品）：

| 自動農場 | 自動種植/收穫 | ⭐ |
