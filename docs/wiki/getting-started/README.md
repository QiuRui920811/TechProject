# 🚀 新手入門

歡迎來到 科技專案 科技世界！本章節將帶你從零開始，建造第一台機器並踏上科技之路。

---

## 開始之前

科技專案 是一個漸進式科技插件。你需要：

1. **收集原版材料** — 鐵錠、銅錠、紅石、書、講台等
2. **搭建進階工作台** — 在鐵方塊上面放工作台
3. **製作研究台** — 在進階工作台上合成，放下後開始研究
4. **解鎖並建造機器** — 花研究經驗 解鎖，再回進階工作台製作
5. **供給能源** — 機器需要電力才能運作

---

## 核心流程

```
收集原版材料（鐵錠、銅錠、書、講台、紅石）
    ↓
搭建進階工作台（鐵方塊 + 工作台）
    ↓
在進階工作台上製作研究台
    ↓
製作淘金盤（4 鐵錠 + 碗），開始篩礫石收集金屬粉
    ↓
研究並解鎖 太陽能發電機 + 粉碎機
    ↓
回到進階工作台製作它們
    ↓
解鎖合金熔煉爐，用合金材料邁入中階科技！
```

---

## 首批建議解鎖

以下機器在遊戲開始時即可解鎖（初始解鎖物品）：

| 工具/機器 | 用途 | 優先順序 |
|---|---|---|
| 淘金盤 | 手動篩礫石→金屬粉塵 | ⭐⭐⭐ |
| 碎礦機 | 礦石→粉末（雙倍產出） | ⭐⭐⭐ |
| 太陽能發電機 | 白天免費發電 | ⭐⭐⭐ |
| 燃煤發電機 | 燒燃料發電 | ⭐⭐ |
| 電鋸 | 木材加工 | ⭐⭐ |
| 回收機 | 回收多餘物品 | ⭐ |
| 自動農場 | 自動種植/收穫 | ⭐ |
