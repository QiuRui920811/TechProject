# 第一座電網：5 分鐘接好你的第一台機器

新手看到導能線纜、能源節點、電池庫會很懵——別慌，這頁會用最簡單的流程，**帶你從零搭一個能跑的電爐**。

---

## TL;DR

> **最簡配方**：太陽能發電機 + 導能線纜 ×2 + 能源節點 + 電爐 + 電池庫
>
> **擺法**：發電機 → 線纜 → 線纜 → 能源節點（中心），電爐和電池庫放節點旁邊（5 格內）即可。

---

## 步驟 1：準備材料

在工作台合成以下物品（前置：完成 [第一步：搭建進階工作台](first-steps.md)）：

| 物品 | 數量 | 用途 |
|---|---|---|
| 太陽能發電機 | 1 | 發電來源 |
| 導能線纜 | 2~3 | 連接發電機與節點 |
| 能源節點 | 1 | **關鍵元件**：周圍 5 格自動供電 |
| 電池庫 | 1（可選） | 儲電備用 |
| 電爐 | 1 | 你的第一台用電機器 |

> 💡 沒有能源節點？先在工作台合成——4 個鐵錠 + 1 個訊號中繼器即可。

---

## 步驟 2：擺設（俯視圖）

找一塊露天平地，按以下配置擺放：

```
   空     空     空     空     空
   空   ⚡發電機  🟦線纜  🟦線纜  ⚡節點    ← 主軸
   空     空     空     空    🔥電爐  ← 節點旁 = 通電
   空     空     空   🔋電池   空    ← 節點旁 = 通電
   空     空   ⚙粉碎機  空     空    ← 節點 2 格內 = 通電
```

### 要點

1. **太陽能發電機**放最外側，**上方不要蓋方塊**（要曬得到太陽）
2. **線纜**把發電機和節點串起來（1~3 格都可以，記得水平相鄰）
3. **能源節點**是整個電網的中心——接下來所有機器都圍著它擺
4. **機器**放在節點的 **5 格範圍內**任意位置（不用貼線纜、不用貼節點，**隨便放就好**）

---

## 步驟 3：驗證通電

1. 等白天（太陽能才會運作）
2. **對機器右鍵**打開 GUI
3. 左上角看到能量條有在上升 → 電通了 ✅
4. 放材料進輸入欄、開始加工 → 你的第一台科技機器上線了！

---

## 常見問題

### ❓ 機器沒電？

**檢查清單**：
1. 發電機上方有沒有被方塊遮擋？太陽能需要露天
2. 線纜有沒有**水平相鄰**串起來？斜角不算連接
3. 機器是不是在能源節點的 **5 格範圍內**？超過就吃不到電了
4. 是不是晚上？太陽能白天才發電，夜晚請搭配**燃煤發電機**或**電池庫**

### ❓ 想拉遠距離？

- 線纜有 **8 格傳輸上限**（每段線纜算 1 格深度）
- 想拉更遠：**每 8 格加一個能源節點**，節點會重置深度（當信號放大器用）
- 節點之間用線纜串，各自的 5 格範圍覆蓋各自工廠區塊

```
發電機 ─線─線─線─ 節點A ─線─線─線─ 節點B ─線─線─線─ 節點C
                  ↓              ↓              ↓
              區域 A 機器     區域 B 機器    區域 C 機器
```

### ❓ 機器太多 1 個節點不夠？

- 節點的 5 格範圍大約能涵蓋 **20~30 台機器**（11×11×11 立方體）
- 超過就再放一個節點，兩個節點**之間用 1 格線纜連一下**就能互通
- 或直接讓兩個節點的**輻射範圍重疊**也行

---

## 進階：混搭線纜直連

老手如果想要精細控制：
- 機器**直接貼著線纜**也能吸電（老行為保留）
- 可以用線纜「指定電流走向」，避開不想供電的機器
- 新方式（節點輻射）和老方式（貼線）**可以混用**，不衝突
