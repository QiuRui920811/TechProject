# 護甲套裝

TechMC 有 4 套科技護甲和 4 套行星防護裝。

---

## 泰坦套裝 (二階)

基礎科技護甲，在製造機中合成。

| 部件 | 材料 |
|---|---|
| 泰坦頭盔 | 複合面板 + 電路板 |
| 泰坦胸甲 | 複合面板 + 碳纖網 |
| 泰坦護腿 | 複合面板 + 銅線 |
| 泰坦戰靴 | 複合面板 + 橡膠片 |

### 所需前置
- 壓縮器 → 複合面板 (鐵板 + 碳纖網)
- 製造機 → 電路板 (矽 + 銅錠)

---

## 量子套裝 (三階)

高階科技護甲，在量子處理器中合成。

| 部件 | 材料 |
|---|---|
| 量子頭盔 | 量子晶片 + 鈦合金 |
| 量子胸甲 | 量子晶片 + 光子板 |
| 量子護腿 | 量子晶片 + 碳纖網 |
| 量子戰靴 | 量子晶片 + 重力子透鏡 |

### 所需前置
- 量子處理器解鎖
- 進階電路 + 奈米塗層 → 量子晶片
- 光子編織機 → 光子板 (量子胸甲)

## 行星防護套裝

為行星探索設計的專用套裝，在行星鍛造台中合成。

### 通用基底

所有行星套裝需要：
- **外骨骼框架** (異星合金 + 複合面板)
- **密封織層** (合成纖維 + 虛空花)

| 部件 | 額外材料 |
|---|---|
| 頭盔 | 輻射網層 + 玻璃板 |
| 胸甲 | 輻射網層 + 碳纖網 |
| 護腿 | 輻射網層 + 絕緣線材 |
