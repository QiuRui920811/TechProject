# 戰鬥裝備

TechProject 提供完整的武器、護甲、護符和工具系統。

---

## 裝備總覽

| 類型 | 數量 | 階級範圍 |
|---|---|---|
| [武器](weapons.md) | 10 種 | TIER 3-4 |
| [護甲](armor.md) | 4 套 (16 件) | TIER 2-4 |
| [護符](talismans.md) | 12 種 | TIER 3 |
| [工具](tools.md) | 5 種 | TIER 2-3 |

---

## 裝備取得

所有戰鬥裝備需要在特定機器中合成：

| 機器 | 可製造裝備 |
|---|---|
| 製造機 | 泰坦套裝、工具 |
| 雷射雕刻機 | 高階護符 (6 種) |
| 力場鍛造台 | TIER3 武器 (5 種)、基礎護符 (6 種) |
| 量子處理器 | 量子套裝 |
| 物質編譯器 | TIER4 武器 (5 種)、虛空套裝 |
| 行星鍛造台 | 行星防護套裝 (4 套) |

---

## 裝備階級

```
TIER 2: 泰坦套裝 (基礎護甲)
   ↓
TIER 3: 量子套裝 + TIER3 武器 + 護符
   ↓        ↓
   │   行星防護套裝 (行星探索必備)
   ↓
TIER 4: 虛空套裝 + TIER4 武器 (最強裝備)
```
