# 武器

科技武器分為 三階和四階 兩個等級。

---

## 三階武器

在力場鍛造台中合成，需要進階材料。

| 武器 | 材料 |
|---|---|
| 脈衝法杖 | 進階電路 + 高密碳塊 |
| 風暴法杖 | 進階電路 + 熱線圈 |
| 重力法杖 | 重力子透鏡 + 進階電路 |
| 曲速球 | 量子晶片 + 重力子透鏡 |
| 冰晶權杖 | 冷卻電池 + 進階電路 |

### 武器說明

**脈衝法杖**
- 發射脈衝能量波
- 基礎遠程武器

**風暴法杖**
- 召喚閃電攻擊
- 範圍傷害

**重力法杖**
- 控制重力場
- 牽引/擊退敵人

**曲速球**
- 投擲型武器
- 造成曲速空間扭曲

**冰晶權杖**
- 發射冰凍射線
- 減速敵人

---

## 四階武器

在物質編譯器中合成，需要最高階材料。

| 武器 | 材料 |
|---|---|
| 電漿長矛 | 電漿電池 + 鈦合金 |
| 虛空之鏡 | 光子板 + 重力子透鏡 |
| 時間膨脹器 | 量子晶片 + 控制單元 |
| 修復信標 | 數據核心 + 聚變核心 |
| 熵變權杖 | 虛空信標核心 + 電漿電池 |

### 武器說明

**電漿長矛**
- 高傷害近戰武器
- 電漿能量附著

**虛空之鏡**
- 反射型防禦武器
- 反彈敵人攻擊

**時間膨脹器**
- 減緩時間流速
- 區域控場

**修復信標**
- 範圍治療裝置
- 隊友回復生命

**熵變權杖**
- 範圍 6 格內對所有非玩家生物造成 8 傷害 + 凋零二 4 秒
- 受領地保護限制，在他人領地內不會生效
- 需要虛空信標核心
