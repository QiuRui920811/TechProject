# 工具

科技工具提供特殊的功能性裝備。

---

## 工具列表

### 科技磁石

| 屬性 | 值 |
|---|---|
| 階級 | 二階 |
| 機器 | 製造機 |
| 材料 | 磁環 + 能量電池 + 銅線 |
| 功能 | 自動吸取周圍掉落物 |

放在背包中即可自動吸取周圍 **5 格**內的掉落物品。手持磁石**蹲下 + 右鍵**可切換開關。

---

### 田野鐮刀

| 屬性 | 值 |
|---|---|
| 階級 | 二階|
| 機器 | 製造機 |
| 材料 | 收割單元 + 鋼板 |
| 功能 | 3×3 範圍收穫作物 |

手持田野鐮刀右鍵點擊作物，可一次收穫 3×3 範圍內的所有成熟作物。

---

### 水培鏟

| 屬性 | 值 |

右鍵點擊農作物使其快速生長。

---

### 向量抓鉤

| 屬性 | 值 |
|---|---|
| 階級 | 二階|
| 機器 | 製造機 |
| 材料 | 伺服模組束 + 絕緣線材 + 感測陣列 |
| 功能 | 鉤爪移動 |

發射鉤爪到目標方塊，將玩家拉向該位置。適合快速移動和探索。

---

### 脈衝推進器

### 風暴噴射背包

| 屬性 | 值 |
|---|---|
| 階級 | 三階|
| 機器 | 行星鍛造台 |
| 材料 | 前線核心 + 風暴導流層 + 無人機框架 + 精密噴嘴 |

  進階製造機 → 脈衝推進器
