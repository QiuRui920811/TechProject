## 工具列表

### 科技扳手

| 屬性 | 值 |
|---|---|
| 階級 | 二階 |
| 機器 | 合成器 |
| 材料 | 鐵板 + 銅錠 |
| 能源 | 12 J |
| 功能 | 快速拆卸科技方塊 |

| 操作 | 效果 |
|---|---|
| **右鍵** 機器 | 啟動（開機） |
| **左鍵** 機器 | 暫停（關機） |
| **蹲下 + 右鍵** 機器 | 拆除並回收 |

拆卸時會保留機器儲存的能源，且不消耗耐久。物品優先放入背包，背包滿時才會掉落地上。

> ⚠️ 拆卸有 **500ms 冷卻時間**，防止快速連點誤拆其他機器。
>
> 可透過 `/tech wrench get` 指令補發（限管理員）。
