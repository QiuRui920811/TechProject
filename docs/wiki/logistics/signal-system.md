# 信號系統

信號系統是科技插件的第三大流動網路（能源、物品、**信號**），讓玩家用簡單的元件組合出複雜的自動化控制邏輯。

---

## 設計理念

信號系統遵循「**樂高/紅石哲學**」——每個元件只做一件事，但組合起來能實現開發者從未預想的玩法。

- 信號強度 **0-15**（與紅石一致）
- 信號**不會隨距離衰減**，範圍由 BFS 深度限制（基礎 8 格 + 範圍升級每階 +6）
- 可透過**紅石轉接器**與原版紅石雙向橋接

---

## 信號元件一覽

### 傳輸

| 元件 | 方塊外觀 | 耗電 | 功能 |
|---|---|---|---|
| 信號線 | 鎖鏈 | 0 | 傳輸信號，連接信號設備（等同物流導管的信號版） |

### 輸入/輸出

| 元件 | 方塊外觀 | 耗電 | 功能 |
|---|---|---|---|
| 信號發射器 | 標靶 | 1 EU/t | 讀取相鄰機器狀態 → 輸出信號 0-15 |
| 信號接收器 | 伏態感測器 | 1 EU/t | 接收信號 → 控制相鄰機器開/關 |

### 邏輯

| 元件 | 方塊外觀 | 耗電 | 功能 |
|---|---|---|---|
| 信號反轉器 | 氧化銅燈 | 0 | NOT 閘：輸出 = 15 - 輸入 |
| 信號比較器 | 銅燈 | 1 EU/t | 比較模式（A≥B→A，否則 0）或 相減模式（A-B） |

### 時序

| 元件 | 方塊外觀 | 耗電 | 功能 |
|---|---|---|---|
| 信號計時器 | 鑿製銅 | 1 EU/t | 週期性脈衝（可設 1-60 秒間隔） |
| 信號計數器 | 校頻伏態感測器 | 1 EU/t | 計算輸入信號上升沿次數（輸出 = 計數 mod 16） |

## 信號發射器模式

在 GUI 中點擊切換讀取模式：

| 模式 | 說明 | 輸出 |
|---|---|---|
| 能量等級 | 讀取相鄰機器能量百分比 | `storedEnergy × 15 / maxCapacity` |
| 物品數量 | 讀取相鄰機器輸入格佔用數 | 非空格數按比例映射到 0-15 |
| 運行狀態 | 機器是否正在加工 | 運行中 = 15，否則 = 0 |
| 輸出阻塞 | 機器輸出是否已滿 | 阻塞中 = 15，否則 = 0 |

---

## 信號接收器設定

| 設定 | 說明 |
|---|---|
| 模式 | **正向**：信號 ≥ 閾值 → 開啟目標；**反向**：信號 < 閾值 → 開啟目標 |
| 閾值 | 0-15，預設 1。左鍵 +1 / 右鍵 -1 |

---

## 信號比較器模式

| 模式 | 邏輯 |
|---|---|
| 比較 | A ≥ B → 輸出 A；否則輸出 0 |
| 相減 | 輸出 = max(A - B, 0) |

A 路與 B 路由**信號輸入方向**決定。

## 應用範例

### 基礎：自動暫停

當電爐能量不足時暫停碎礦機，避免浪費電力：

```
碎礦機 ←[信號接收器]←[信號線]←[信號發射器]→ 電爐
                                    (能量等級模式)
```

- 發射器讀取電爐能量等級
- 接收器設定閾值 = 3（能量低於 ~20% 時關閉碎礦機）

### 中階：脈衝批次加工

用計時器控制機器週期性開關：

```
機器 ←[信號接收器]←[信號線]←[信號計時器]
                               (間隔 = 5s)
```

### 進階：條件邏輯

用比較器判斷兩條產線的負載差異：

```
產線A [發射器] ──→ [比較器] ──→ [接收器] → 分流閥
產線B [發射器] ──↗   (相減模式)
```

### 終極：紅石整合

用原版紅石拉桿控制科技機器：

```
拉桿 → 紅石線 → [紅石轉接器(輸入)] → [信號線] → [信號接收器] → 機器
```

用科技信號控制原版紅石燈：

```
[信號發射器] → [信號線] → [紅石轉接器(輸出)] ─▶ 紅石燈 / 紅石線
                             ↑
                       方塊自身發 0-15 紅石
```

## 紅石轉接器

雙向橋接原版紅石與科技信號。方塊外觀是 **標靶（TARGET）**，本身就是可變強度的紅石源。

| 模式 | 說明 |
|---|---|
| 輸入 (紅石→信號) | 讀取方塊紅石功率 (0-15) → 輸出為科技信號 |
| 輸出 (信號→紅石) | 讀取科技信號 → **方塊自身**發出對應強度紅石 (0-15) |

在 GUI 中點擊切換模式。輸出模式下，**轉接器旁邊直接貼紅石線 / 比較器 / 紅石燈都能讀到訊號**，不需要放額外的紅石火把。

> 💡 比較器模式可以讀到連續 0-15 強度，適合做類比控制；紅石線接普通開關則是 0 = 關、> 0 = 開。

## 解鎖條件

所有信號機器需要先合成**信號核心**（需要訊號中繼器），紅石轉接器額外需要**紅石介面**。

```
訊號中繼器（已有）
  └→ 信號核心
      ├→ 信號線 / 發射器 / 接收器 / 反轉器
      ├→ 信號比較器（+高級電路）
      ├→ 信號計時器（+伺服馬達）
      └→ 信號計數器（+高級電路）
  └→ 紅石介面
      └→ 紅石轉接器
```
