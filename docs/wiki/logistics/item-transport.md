# 物品傳輸

使用物流導管和物流節點建立自動物品搬運網路。

---

## 傳輸元件

### 物流導管

| 屬性 | 值 |
|---|---|
| 階級 | 一階|
| 合成 | 工作台 (壓力導管 ×3) |
| 功能 | 物品傳輸通道 |

將物流導管連接在機器之間，形成物品傳輸路徑。

### 物流節點

| 屬性 | 值 |
|---|---|
| 階級 | 二階|
| 合成 | 工作台 (鐵錠 ×4 + 訊號中繼器) |
| 功能 | 物品分配中樞 |

物流節點是物品傳輸網路的核心，可以連接多條物流管路。

---

## 網路配置

> 💡 採集機器可以直接把物品推送到相鄰機器；但如果是**加工機器 → 加工機器**，仍然必須透過**物流導管**、**物流節點**或物流終端（如倉儲匯流站、過濾路由器）做中繼，直接貼鄰不會自動傳輸。

### 點對點傳輸

```
碘礦機 ──[物流導管]──→ 電爐
```

將一台機器的輸出透過導管送到另一台。若來源與目標都是加工機器，這也是必要條件，不是可選優化。

### 中樞傳輸

```
碎礦機 ──┐
電鋸   ──┤──→ 物流節點 ──→ 電爐
自動農場──┘              ──→ 壓縮器
                         ──→ 製造機
```

使用物流節點作為中心，連接多台機器。

## 分流元件

### 分流節點

| 屬性 | 階級 | 功能 |
|---|---|---|
| 分流節點 | 二階| 將物品平均分配到多條路徑 |

### 過濾路由器

| 屬性 | 階級 | 功能 |
|---|---|---|
| 過濾路由器 | 二階 | 根據物品類型過濾分配 |

#### 使用方式

- 前 **1~3 格**放入「樣本物品」定義過濾規則（最多 3 種）
- 後面的 input 格（slot 4~9）才是流入要被過濾的物品
- GUI 切換 **白名單**（只放樣本）／**黑名單**（放樣本以外的）

#### ⚠️ 常見錯誤擺法

**陷阱 A：樣本中間空格**

樣本必須**從第 1 格連續放**，遇到空格就停止讀取後面的格子。

```
❌ 錯誤：[鐵粉] [空] [銅粉]
        ↑      ↑     ↑
        樣本    停！   被當成要過濾的物品 → 反而被路由出去

✅ 正確：[鐵粉] [銅粉] [鉛粉]
        三種都被當作樣本
```

**陷阱 B：樣本來源不對**

樣本必須是**機器產出的科技物品**，不能用 `/give` 拿原版染料替代：

| 物品 | 看起來 | 電腦識別 |
|---|---|---|
| 機器產的銅粉 | 橘色染料外觀 | `copper_dust`（科技 ID） |
| 原版橘色染料 | 一模一樣 | `orange_dye`（原版 ID） |
| 機器產的鉛粉 | 灰色染料外觀 | `lead_dust`（科技 ID） |
| 原版灰色染料 | 一模一樣 | `gray_dye`（原版 ID） |

兩個 ID 不匹配 → 過濾器找不到對應物品 → 銅粉/鉛粉過不出來。

**最簡單的解法**：把實際要過濾的那個物品複製一份放樣本格，100% 對得上。

## 進階分流元件

### 優先路由器

| 屬性 | 階級 | 功能 |
|---|---|---|
| 優先路由器 | 三階| 依過濾欄設定優先物品，匹配項走主路、其餘走備路 |

使用方式與過濾路由器相似，前 1~3 格放入樣本物品定義「優先物品」。

合成：優先級模組 + 分流核心 + 訊號中繼器×2 → 製造機

### 輪流分配器

| 屬性 | 階級 | 功能 |
|---|---|---|
| 輪流分配器 | 三階| 平均分配物品到多台下游機器（Round-Robin） |

與分流節點類似，但嚴格輪流——確保每台下游機器收到相同數量的物品。

合成：分配齒輪 + 分流核心 + 訊號中繼器×2 → 製造機

### 溢出處理器

| 屬性 | 階級 | 功能 |
|---|---|---|
| 溢出處理器 | 二階| 主路滿載時自動轉送備用路線 |

優先向主路（輸出方向）推送物品；當主路下游全滿時，自動切換到備用路線。適合搭配垃圾節點或儲存中心做防堵塞。

合成：溢出閥 + 分流核心 → 製造機

### 漏斗介面

| 屬性 | 階級 | 功能 |
|---|---|---|
| 漏斗介面 | 二階| 原版漏斗/箱子 ↔ 科技機器的物品橋接 |

放在科技機器旁邊，自動：
- 從相鄰原版容器抽取物品 → 推入科技機器輸入
- 從科技機器輸出 → 推入下方原版容器

速度與原版漏斗相同（每 4 tick 搬運 1 個物品）。

合成：漏斗轉接模組 + 機械零件 → 製造機

---

## 應用範例

### 自動碎礦冶煉線

```
碎礦機 ──[物流導管]──→ 過濾路由器
                          ├──→ 電爐 (粉末→錠)
                          ├──→ 壓縮器 (鐵錠→鐵板)
                          └──→ 垃圾節點 (多餘材料)
```

### 農業生產線

```
自動農場 ──→ 物流節點 ──→ 生質實驗室 (作物→食物)
                       ──→ 洗礦機 (作物→粉碎生質)
                       ──→ 儲存中心 (存放多餘)
```
