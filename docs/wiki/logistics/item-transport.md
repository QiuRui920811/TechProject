# 物品傳輸

使用物流導管和物流節點建立自動物品搬運網路。

---

## 傳輸元件

### 物流導管

| 屬性 | 值 |
|---|---|
| 階級 | 一階|
| 合成 | 工作台 (壓力導管 ×3) |
| 功能 | 物品傳輸通道 |

將物流導管連接在機器之間，形成物品傳輸路徑。

### 物流節點

| 屬性 | 值 |
|---|---|
| 階級 | 二階|
| 合成 | 工作台 (鐵錠 ×4 + 訊號中繼器) |
| 功能 | 物品分配中樞 |

物流節點是物品傳輸網路的核心，可以連接多條物流管路。

---

## 網路配置

> 💡 採集機器可以直接把物品推送到相鄰機器；但如果是**加工機器 → 加工機器**，仍然必須透過**物流導管**、**物流節點**或物流終端（如倉儲匯流站、過濾路由器）做中繼，直接貼鄰不會自動傳輸。

### 點對點傳輸

```
碘礦機 ──[物流導管]──→ 電爐
```

將一台機器的輸出透過導管送到另一台。若來源與目標都是加工機器，這也是必要條件，不是可選優化。

### 中樞傳輸

```
碎礦機 ──┐
電鋸   ──┤──→ 物流節點 ──→ 電爐
自動農場──┘              ──→ 壓縮器
                         ──→ 製造機
```

使用物流節點作為中心，連接多台機器。

---

## 分流元件

### 分流節點

| 屬性 | 階級 | 功能 |
|---|---|---|
| 分流節點 | 二階| 將物品平均分配到多條路徑 |

### 過濾路由器

| 屬性 | 階級 | 功能 |
|---|---|---|
| 過濾路由器 | 二階| 根據物品類型過濾分配 |

### 垃圾節點

| 屬性 | 階級 | 功能 |
|---|---|---|
| 垃圾節點 | 二階| 銷毀不需要的物品 |

---

## 應用範例

### 自動碎礦冶煉線

```
碎礦機 ──[物流導管]──→ 過濾路由器
                          ├──→ 電爐 (粉末→錠)
                          ├──→ 壓縮器 (鐵錠→鐵板)
                          └──→ 垃圾節點 (多餘材料)
```

### 農業生產線

```
自動農場 ──→ 物流節點 ──→ 生質實驗室 (作物→食物)
                       ──→ 洗礦機 (作物→粉碎生質)
                       ──→ 儲存中心 (存放多餘)
```
