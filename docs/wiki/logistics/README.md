# 物流系統

科技專案 的物流系統讓你建立自動化的能源傳輸和物品運輸網路。

---

## 系統構成

| 系統 | 說明 | 頁面 |
|---|---|---|
| [能源網路](energy-network.md) | 發電、傳輸、儲存能源 | 導能線纜、能源節點、電池庫 |
| [物品傳輸](item-transport.md) | 自動搬運物品 | 物流導管、物流節點 |
| [貨物系統](cargo-system.md) | 進階物流管理 | 貨物管理器、過濾路由器 |

---

## 物流一覽

```
          能源網路
發電機 ──[導能線纜]──[能源節點]──→ 機器
                        ↕
                     電池庫

          物品傳輸
機器輸出 ──[物流導管]──[物流節點]──→ 機器輸入
                         ↕
                   過濾路由器 / 分流節點

          貨物系統
貨物輸入節點 ──[貨物馬達]──→ 貨物管理器
                              ↕
                         儲存中心
```

---
