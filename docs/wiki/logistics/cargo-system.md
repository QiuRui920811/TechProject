# 貨物系統

進階物流管理系統，適用於大規模自動化工廠。

---

## 貨物元件

| 元件 | 階級 | 功能 |
|---|---|---|
| 貨物輸入節點 | 二階| 從箱子/機器擷取物品進入貨物網路 |
| 貨物管理器 | 三階| 管理整個貨物網路的物品分配 |
| 貨物馬達 | 二階| 驅動貨物網路運轉 |
| 工業匯流排 | 二階| 大量物品傳輸通道 |
| 儲存中心 | 二階| 數位化儲存大量物品 |
| 過濾路由器 | 二階| 根據物品類型過濾分配 |
| 分流節點 | 二階| 平均分配物品 |
| 垃圾節點 | 二階| 銷毀多餘物品 |

---

## 網路架構

### 基礎貨物網路

```
箱子 ──[貨物輸入節點]──[貨物馬達]──→ 貨物管理器
                                         ↓
                                    儲存中心
```

### 完整工廠網路

```
碎礦機群 ──┐
電爐群   ──┤
壓縮器群 ──┤──→ 貨物輸入節點 ──→ 貨物馬達
製造機群 ──┘                        ↓
                              貨物管理器
                           ┌────┼────┐
                           ↓    ↓    ↓
                      過濾  儲存  垃圾
                      路由器 中心  節點
                        ↓
                 ┌──────┼──────┐
                 ↓      ↓      ↓
              機器A  機器B  機器C
```

---

## 儲存中心

| 屬性 | 值 |
|---|---|
| 階級 | 二階|
| 功能 | 數位化儲存，容量遠超普通箱子 |
| 特點 |介面 介面查看所有存放物品 |

### 使用要點

- 可連接到貨物管理器自動存取
- 多台儲存中心擴展容量
- 支援搜尋功能快速找到物品

---

## 貨物管理器

| 屬性 | 值 |
|---|---|
| 階級 | 三階|
| 功能 | 管理整個貨物網路 |
| 特點 | 設定物品路由規則 |

### 路由規則

- 可設定哪些物品送往哪些目的地
- 支援優先順序設定
- 多餘物品自動送往儲存中心

---

## 建設建議

### 小型工廠

```
5-10 台機器 → 1 個貨物管理器 + 1 個儲存中心
```

### 中型工廠

```
10-30 台機器 → 1 個貨物管理器 + 3-5 個儲存中心
              + 過濾路由器分流
```

### 大型工廠

```
30+ 台機器 → 多個貨物管理器分區管理
             + 工業匯流排連接各區
             + 大量儲存中心
```
