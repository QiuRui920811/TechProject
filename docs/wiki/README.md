# 🔬 TechProject 科技百科

> **TechProject** 是一款 Minecraft 科技模組插件，提供完整的工業自動化、星際探索、戰鬥裝備與烹飪系統。從第一台碎礦機起步，一路打造歐米茄核心，征服五大星球！

---

## ✨ 特色系統

| 系統 | 說明 |
|---|---|
| ⚙️ **75+ 台機器** | 從碎礦機到巔峰鍛台，橫跨 4 個科技階級 |
| 🧱 **250+ 種物品** | 材料、元件、合成品、食物、裝備一應俱全 |
| 📖 **200+ 道配方** | 機器配方 + 工作台藍圖 + 互動烹飪 |
| 🪐 **5 顆星球** | 各有獨特環境危害與專屬資源 |
| ⚔️ **10 種武器** | 從脈衝法杖到熵變權杖 |
| 🛡️ **4 套護甲 + 4 套行星防護** | 泰坦、量子、虛空套裝 + 星球專屬防護 |
| 🔮 **12 種護符** | 被動效果加持 |
| 🍳 **60+ 種食物** | 機器合成 + 營火烹飪 + 飲品調製 |
| 🏆 **67 項成就** | 每項對應可穿戴稱號 |
| 🤖 **安卓機器人** | 可編程的自動化工人 |
| 🔌 **物流網路** | 能源傳輸 + 物品管道 + 貨物系統 |
