# 系統機器

大型自動化設備，用於大規模資源採集和管理。

---

## 採礦系統

### 礦場鑽機

| 型號 | 階級 | 範圍 | 說明 |
|---|---|---|---|
| 礦場鑽機 Mk1 | 二階| 小型 | 基礎自動採礦 |
| 礦場鑽機 Mk2 | 二階| 中型 | 進階效率 |
| 礦場鑽機 Mk3 | 三階| 大型 | 最大採礦範圍 |

#### 所需元件

| 元件 | 來源 |
|---|---|
| 鑽頭 | 系統配方 |
| 礦場框架 | 系統配方 |

#### 注意事項

- 黑名單方塊無法挖掘：基岩、屏障、指令方塊等
- 受範圍升級影響
- 需要持續供電

## 收集系統

### 真空吸入口

| 屬性 | 值 |
|---|---|
| 階級 | 二階|
| 功能 | 自動拾取範圍內的掉落物品 |
| 元件 | 真空核心 |

### 生物收集器

| 屬性 | 值 |
|---|---|
| 階級 | 二階|
| 功能 | 自動收集範圍內怪物掉落物 |
| 元件 | 捕蟲器 |

#### 支援生物

| 類別 | 生物 |
|---|---|
| 被動 | 牛、羊、豬、雞、兔子 |
| 常見怪物 | 僵屍、骷髏、苦力怕、蜘蛛、史萊姆、屍殼、流浪者、溺屍 |
| 地獄生物 | 烈焰人、岩漿怪、惡魂、凋靈骷髏、僵屍豬布林、豬布林、豬布林蠻兵、豬布獸、豬屍獸、熾足獸 |
| 生電常用 | 鐵傀儡、女巫、末影人 |
| 突襲生物 | 掠奪者、衛道士、喚魔者、劫毀獸 |
| 海底生物 | 守衛者、遠古守衛者 |

#### 生物黑名單

以下生物不受影響：玩家、盔甲架、村民、流浪商人、命名生物、馴服生物、拴繩生物、幼年生物等。
