## 自動化等級

### 等級一：基礎自動化

```
自動農場 → 作物自動種植收穫
太陽能發電機 → 自動發電
```

### 等級 1.5：純電力版機器

```
手動機器 + 自動化模組 → 純電力版
  ├── 電動粉碎機 → 全自動粉碎
  ├── 電動壓縮機 → 全自動壓縮
  ├── 電動洗礦機 → 全自動洗礦
  ├── 電動拉線機 → 全自動拉線
  ├── 電動淨化器 → 全自動淨化
  ├── 電動離心機 → 全自動離心
  ├── 電動生質實驗室 → 全自動生質培養
  └── 電動化學反應器 → 全自動化學合成
```

> 耗能約手動版 2×，但完全不需手動右鍵觸發。詳見 [純電力版機器](../machines/electric-machines.md)

### 等級二：系統自動化

```
礦場鑽機 → 自動挖礦
作物收割器 → 大範圍收割
伐木機 → 自動伐木
釣魚碼頭 → 自動釣魚
風暴渦輪 → 風力發電
```

### 等級三：完全自動化

```
安卓機器人 → AI 工人
  ├── 農務程序 → 自動種植/收穫
  ├── 伐木程序 → 自動伐木
  ├── 回收程序 → 自動回收
  └── 行星採樣程序 → 行星資源採集
```
