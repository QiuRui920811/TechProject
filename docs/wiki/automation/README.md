## 自動化等級

### 等級一：基礎自動化

```
自動農場 → 作物自動種植收穫
太陽能發電機 → 自動發電
```

### 等級二：系統自動化

```
礦場鑽機 → 自動挖礦
作物收割器 → 大範圍收割
伐木機 → 自動伐木
釣魚碼頭 → 自動釣魚
風暴渦輪 → 風力發電
```

### 等級三：完全自動化

```
安卓機器人 → AI 工人
  ├── 農務程序 → 自動種植/收穫
  ├── 伐木程序 → 自動伐木
  ├── 回收程序 → 自動回收
  └── 行星採樣程序 → 行星資源採集
```
