# 🐔 基因雞工程

基因雞工程是一套獨立的資源生產系統。透過捕捉野生雞、分析基因、選擇性繁殖，最終培育出能持續產出稀有資源的特殊品種雞。

> 此系統屬於 **農業 / 生質** 體系（AGRI_BIO），所有物品與機器歸類在科技百科的「農業 / 果園」分頁。

---

## 快速入門流程

```
合成雞網 → 捕捉野雞 → 基因定序器分析 DNA → 私人雞舍繁殖 → 激發室持續產出資源
```

1. 在**合成器**製作「雞網」
2. 手持雞網 **右鍵野生雞** → 獲得一隻「口袋雞」（顯示為雞頭顱）
3. 將口袋雞放入**基因定序器** → 揭示 DNA 序列
4. 挑選合適親本放入**私人雞舍** → 繁殖出帶有更多隱性基因的後代
5. 將目標品種放入**激發室** → 持續產出對應資源

---

## 基因系統

每隻雞擁有 **6 個基因位點**（B、C、D、F、S、W），每個位點由兩個等位基因組成：

| 符號 | 含義 | 範例 |
|---|---|---|
| 大寫 (如 B) | 顯性 | BB = 同型顯性 |
| 小寫 (如 b) | 隱性 | bb = 同型隱性 |
| 大小混合 | 異型合子 | Bb = 攜帶隱性 |

DNA 以 12 字元顯示，例如 `[Bb] [CC] [Dd] [FF] [SS] [Ww]`。

簡碼型用 3 種字母表示每個位點的狀態：
- **D** = 同型顯性（兩個都是大寫）
- **H** = 異型合子（一大一小，攜帶隱性但不表現）
- **r** = 同型隱性（兩個都是小寫）✅ 目標

### 核心規則

- **同型隱性 (r) 的位點數量 = 資源等級**（0 ～ 6 級）
- 隱性位點越多 → 資源越稀有，激發室產出速度也越快
- 繁殖採用**孟德爾遺傳**：後代每個位點從父母各隨機取一個等位基因

### 繁殖技巧

- 野生雞大多為顯性基因，少數位點會帶有隱藏的隱性等位基因
- 透過定序器確認誰帶有隱性，再讓帶有相同位點隱性的雞配對
- 例如：父方 `[Bb]` × 母方 `[Bb]` → 有 **25%** 機率出現 `[bb]`（同型隱性）
- 耐心選育多代，逐步累積越來越多的同型隱性位點

## 物品

### 雞網

| 屬性 | 值 |
|---|---|
| 圖示 | 牽繩 |
| 取得 | 合成器（線 ×2 + 木棍 → 雞網 ×4，3 J） |

右鍵野生雞使用，一次性消耗。雞會被移除，你會獲得一隻未定序的口袋雞。

### 口袋雞

| 屬性 | 值 |
|---|---|
| 圖示 | 雞頭顱 |
| 取得 | 雞網捕捉野生雞 |

被捕獲的雞會以雞頭顱形式存在於背包中，帶有隱藏的 DNA。  
經過基因定序器分析後，物品的 lore 會顯示完整基因型、簡碼型、資源等級與對應資源。

## 機器

### 基因定序器

| 屬性 | 值 |
|---|---|
| 方塊 | 切石機 |
| 耗能 | 3 J/tick |
| 合成 | 合成器（玻璃 + 鐵錠 + 紅石，8 J） |
| 解鎖 | 雞網 |

將未定序的口袋雞放入輸入格 → 消耗能量 → 輸出已定序的口袋雞，DNA 資訊顯示在 lore 上。

### 私人雞舍

| 屬性 | 值 |
|---|---|
| 方塊 | 堆肥桶 |
| 耗能 | 2 J/tick |
| 合成 | 合成器（橡木板 + 乾草捆，5 J） |
| 解鎖 | 基因定序器 |

放入 **兩隻已定序的口袋雞** → 消耗能量 → 繁殖出一隻後代口袋雞。  
父母不會被消耗，會留在輸入格中。後代的每個位點遵循孟德爾遺傳，從父母各取一個等位基因。

**繁殖限制：**
- 每次繁殖後有 **120 秒冷卻**，冷卻期間顯示「冷卻 Xs ┃ 壽命 Y/30」
- 每隻雞最多繁殖 **30 次**，用完壽命後會消失

### 激發室

| 屬性 | 值 |
|---|---|
| 方塊 | 釀造台 |
| 耗能 | 4 J/tick |
| 合成 | 合成器（鐵錠 + 玻璃 + 紅石，10 J） |
| 解鎖 | 私人雞舍 |

放入一隻**有同型隱性位點的已定序口袋雞** → 持續消耗能量 → 不斷在輸出格產出對應資源物品。  
隱性位點越多，產出速度越快。

**激發限制：**

| 等級 | 激發週期 | 壽命上限 |
|------|--------|----------|
| Tier 1 | 30 秒 | 1000 次 |
| Tier 2 | 60 秒 | 500 次 |
| Tier 3 | 120 秒 | 200 次 |
| Tier 4 | 180 秒 | 120 次 |
| Tier 5 | 300 秒 | 60 次 |
| Tier 6 | 600 秒 | 30 次 |

- 運行時顯示「激發：XX% ┃ 30s/個 ┃ 壽命 Y/Z」
- 雞用完壽命次數後會消失，等級越高越珍貴

## 常見問答

**Q：野生雞一定是全顯性嗎？**  
A：不是，野生雞有小機率在某些位點帶有隱性等位基因，需要捕捉大量並定序才能找到好基因。

**Q：繁殖的後代一定比父母好嗎？**  
A：不一定。孟德爾遺傳是機率性的，後代可能更好也可能退化。多次繁殖、保留最佳後代是關鍵。

**Q：激發室的雞會死掉嗎？**  
A：會。每隻雞有壽命上限（Tier 1 = 1000 次、Tier 6 = 30 次），用完後雞會消失，需要重新培育。

**Q：可以用物流系統自動取出資源嗎？**  
A：可以，激發室的輸出格支持貨物輸出節點自動抽取。
