# 食物系統

科技專案 新增了完整的農業與烹飪系統，包含種植、機器加工和互動烹飪。

---

## 食物類型

| 類型 | 數量 | 取得方式 |
|---|---|---|
| [農務種植](farming.md) | 30+ 種作物 | 自動農場 / 溫室 / 星球種植 |
| [機器食品](machine-food.md) | 25+ 種料理 | 生質實驗室 / 生物合成槽 / 低溫蒸餾器 |
| [飲料](drinks.md) | 20+ 種飲品 | 冷卻混合器 |
| [互動烹飪](../recipes/cooking.md) | 30 道菜 | 營火 / 煙燻爐 |

---

## 食物效果

食物提供飽食度恢復，部分特殊食物還有額外效果：

| 食物類型 | 效果 |
|---|---|
| 基礎料理 | 恢復飽食度 |
| 科技料理 | 較高飽食度恢復 |
| 星球反食物 | 減輕星球危害效果 |
| 飲料 | 提供短暫增益效果 |

---

## 食物取得流程

### 入門路線
```
自動農場 → 基礎作物 → 互動烹飪 (營火)
```

### 進階路線
```
溫室培育艙 → 種子解鎖 → 進階作物
    ↓
生質實驗室 / 冷卻混合器 → 機器料理 / 飲料
```

### 高階路線
```
基因剪接台 → 基因模板 → 生物合成槽 → 合成食品
星球種植 → 外星水果 → 星球反食物
```

---

## 星球反食物

每顆星球都有對應的特殊食物，可減輕該星球環境造成的負面效果：

| 食物 | 星球 | 效果 |
|---|---|---|
| 輻晶蜜凍 | 奧蕾莉亞 | 減輕輻射 |
| 霜芯暖鍋 | 克里昂 | 減輕低溫 |
| 相位凝露 | 倪克斯 | 減輕真空 |
| 日冕雪酪 | 赫利昂 | 減輕日灼 |
| 雷雲汽泡飲 | 坦佩斯特 | 減輕風暴 |
