## 2026/03/21

### 🤝 新增功能：機器信任共享系統

玩家現在可以將自己的機器共享給其他玩家操作，不再需要管理員介入。

| 指令 | 說明 |
|------|------|
| `/tech trust <玩家>` | 對準機器，將該玩家加入這台機器的信任清單 |
| `/tech untrust <玩家>` | 對準機器，將該玩家從信任清單移除 |
| `/tech trustlist` | 對準機器，查看這台機器的信任清單 |
| `/tech trustall <玩家>` | 一鍵將玩家加入你**所有**機器的信任清單 |
| `/tech untrustall <玩家>` | 一鍵從你**所有**機器移除該玩家 |

- 被信任的玩家可以打開/操作機器，但**不能拆除**
- 每台機器獨立信任清單，重啟後仍保留
- 物流/電網仍只認同一主人的機器互聯

### 🐛 修復

- 修復機器 GUI 中左鍵拖曳均分會造成**物品複製**的重大 bug（原因：Bukkit 取消事件後會自動還原游標，導致物品同時存入機器又回到手上）
- 修復**激發室**與基因定序器配方完全相同的問題（激發室紅石改為電路板）

### �🐔 新增系統：基因雞工程

全新的資源生產系統！透過捕捉野生雞、分析 DNA、選擇性繁殖，培育出能持續產出稀有資源的特殊品種雞。

- **雞網**：右鍵野生雞捕捉為口袋雞（合成器製作：線×2 + 木棍）
- **口袋雞**：被捕獲的雞以雞頭顱形式存在，帶有 6 個基因位點的隱藏 DNA
- **基因定序器**：揭示雞的 DNA 序列（切石機，3 J/tick）
- **私人雞舍**：放入兩隻已定序雞繁殖後代，孟德爾遺傳（堆肥桶，2 J/tick）
- **激發室**：放入有隱性位點的雞持續產出資源（釀造台，4 J/tick）
- 共 **64 種資源雞**，從缽毛到獄髓錠，分 7 個等級（Tier 0–6）
- 詳細說明請見 [**基因雞工程專頁**](chickens/README.md)

- 修復**太陽能陣列**無法放入材料進行合成的問題（星光板、網格模組兩個配方現在可正常運作）

### 🔒 配方鎖定功能

- 新增 **配方鎖定** — 當機器有多個配方使用相同材料時，GUI 中會出現鎖定按鈕
