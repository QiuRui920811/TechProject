## 2026/04/08（晚間更新）

### 🤖 安卓機器人 — 探索程序

- **新程序腳本「探索程序」**：安卓獨有能力，沒有其他機器能替代。
  - 可在**任何世界**運作（包含地獄、末地），每次巡邏約 12% 機率探勘到稀有資源。
  - **主世界深層（Y<16）**：鑽石、綠寶石原礦、粗金、青金石、粗銅。
  - **主世界淺層**：鑽石、綠寶石原礦、紫水晶碎片、粗鐵、粗銅。
  - **地獄**：古代碎片（8%）、石英、金粒、螢光石粉、岩漿球。
  - **末地**：末影珍珠、歌萊果、乕牻殼、終界石。
  - 合成：量子框架×2 + 安卓記憶矩陣×2 → 量子處理器 → 探索程序。
- **安卓工作站世界限制解除**：安卓機器人不再受「允許世界」限制，可在地獄、末地等世界自由部署。

### ⛏️ 鈾燃料自動化

- **新配方**：綠寶石 → 電動粉碎機 → 鈾粉×1（原礦仍為×2）。
  - 搭配安卓探索程序可自動取得綠寶石原礦，實現鈾燃料全自動化生產。

### 🌐 科技百科網站

- **科技書主頁新增「科技百科網站」按鈕**：點擊後在聊天出現可點擊的連結，直接開啟網頁版科技百科。

### 🐛 修正

- **地獄機器限制**：修正生物收集器、作物收割機、伐木機、真空管、釣台等機器在地獄無法放置的問題。現在只有採礦鑽機受世界限制。
- **篩礦機產出遺漏**：砂礫篩選新增鉛粉（14%）、靈魂砂篩選新增矽晶（12%）。
- **新玩家成就級聯**：修正新玩家首次登入時幾乎解鎖所有成就的問題。
- **AI 幫手入口**：科技書主頁新增 AI 科技幫手按鈕，可直接在聊天輸入問題查詢。

## 2026/04/08

### 🔩 錫材料擴展

- **新物品**：錫板（壓縮機）、錫線（拉線機）——錫在電力系統中的用途大幅增加。
- **焊錫合金配方調整**：鉛粉 + ~~鋅粉~~ → 鉛粉 + **錫粉**（更貼近真實焊錫成分）。
- **能源電池配方調整**：~~銅錠~~ + 紅石 → **錫錠** + 紅石（錫製電極）。
- **鋰電池配方調整**：能源電池 + ~~銅錠~~ → 能源電池 + **錫板**（錫製外殼）。
- **訊號中繼器配方調整**：邏輯閘 + ~~銅線~~ → 邏輯閘 + **錫線**（低損耗信號傳輸）。
- 壓縮機 / 拉線機 / 電動壓縮機 / 電動拉線機均新增對應錫材配方。

### 🏆 成就系統

- **工業成就全服廣播**：解鎖成就時，訊息會以類似原版進度的格式顯示給所有線上玩家，滑鼠懸停可查看成就描述。

### 🚀 傳送面板改版

- **珍珠改為背包消耗**：傳送不再需要把珍珠放進機器輸入欄，改為從背包扣除原版末影珍珠×1（不會誤扣科技物品）。
- **3 秒吟唱**：選擇目的地後需靜止 3 秒才會傳送，移動會取消。附帶粒子與音效。
- **任何人都能使用**：不再限擁有者，公共設施也能用。
- **公開 / 私人模式**：擁有者蹲下右鍵可命名、切換公開或私人。公開面板所有玩家都能看到；私人面板僅擁有者與受信任玩家可見。

### 🛡️ 科技物品原版行為修正

- **末影珍珠 / 終界之眼 / 玻璃瓶**：以這些材質為底的科技物品右鍵不再觸發原版行為（投擲、飛出、裝水）。
- **不死圖騰**：科技不死圖騰不再觸發復活效果。
- **村民 / 遊商交易**：手持科技物品右鍵村民不會開啟交易；科技物品無法放入交易格。
- **豬靈以物易物**：豬靈不會撿起掉落的科技物品。

### 🐛 修正

- **傳送面板過濾關機**：已關機的傳送面板不再出現在其他面板的傳送選單中；傳送時也會檢查目的地是否開機。
- **機器防水桶**：水桶與熔岩桶無法在機器方塊上放置液體，避免沖掉機器。

## 2026/04/07

### 🏭 新機器

- **萬用合成站**：聚合所有加工類機器的配方，能耗為原配方的 2 倍，適合中期過渡或臨時生產。合成需要機器外殼、進階電路×2、電路基板×4、鋼板×4。

### ⚡ 能源系統

- **機器待機耗電**：所有加工類機器在通電狀態下會持續消耗約 1/4 滿載電力作為待機成本，電力不足會顯示「缺少電力」並停止運作。
- **機器啟停功能**：所有機器 GUI 右下角新增啟停按鈕（綠色＝運行中、灰色＝已暫停）；暫停後不再消耗電力與材料。

### 🔧 科技扳手

- **啟停功能**：左鍵機器→關機、右鍵機器→開機、蹲下右鍵→拆卸（與原本相同）。
- **多方塊清除修復**：修復扳手拆卸多方塊機器（粉碎機、拉線機等）時，上方的鐵柵欄、壓力板等配件方塊沒被清除的問題。
- **拆卸冷卻**：扳手拆卸冷卻時間 500ms，防止快速連點誤拆其他機器。

### ⚖️ 平衡

- 熔爐鐵粉／銅粉熔煉配方能耗從 16 降為 8 EU。

### 🐛 修復

- 爆破鎬不再破壞容器類方塊（箱子、漏斗、木桶等），避免內容物消失。

### 📝 格式

- 材料數量顯示統一為後綴格式：「物品名×N」取代原本的「N× 物品名」。
