## 2026/03/22

### 🏆 成就自動補發

- 修復玩家已達成條件但成就未解鎖的問題
- 打開成就選單（`/tech achievements`）時，系統會自動掃描所有成就並補發已達成但未解鎖的項目
- 玩家登入時也會延遲 2 秒自動檢查一次，確保離線期間達成的條件不會漏掉

### 🐛 成就統計遺漏大修

修復 **8 個成就統計值從未被追蹤** 的嚴重 bug，導致大量成就永遠無法解鎖：

| 成就 | 統計鍵 | 問題 |
|------|--------|------|
| 合成學徒 | `items_crafted` | 從未遞增 |
| 初始電力 | `generators_placed` | 從未遞增 |
| 首次研究 | `research_spent` | 從未遞增 |
| 冶煉先行 | `smelter_cycles` | 機器 ID 是 smeltery，但查詢 smelter → 永遠 0 |
| 加工里程碑 ×3 | `total_processed` | 從未遞增 |
| 儲存大師 | `storage_units_placed` | 從未遞增 |
| 果園管理員 | `crop_varieties` | 從未遞增 |
| 首次造訪星球 | `planets_visited` | 從未遞增 |
| 巨構建造者 | `megastructures_built` | 從未遞增 |

修復內容：
- 在配方完成時遞增 `items_crafted` 和 `total_processed`（手動 + 自動管線）
- 在放置機器時根據類型遞增 `generators_placed`、`storage_units_placed`、`megastructures_built`
- 在研究科技時遞增 `research_spent`
- 在抵達星球時遞增 `planets_visited`
- 修正 `smelter_cycles` → `smeltery_cycles` 的名稱不匹配
- 所有成就條件新增**回溯判定**：基於玩家已解鎖的物品/機器數量推斷歷史成就，現有玩家重新登入或開選單即可自動補發

### 📦 貨物輸出節點

- 新增 **貨物輸出節點** `cargo_output_node` — 自動將產線物品推送到相鄰原版容器（箱子/桶/漏斗/熔爐等）
- 與 Slimefun4 的 Cargo Output Node 功能對齊，物流系統現在雙向都支援原版容器
- 可透過 outputDirection 設定推送方向（預設 ALL = 六向）
- 每 tick 最多推送 4 件物品
- 配方：路由晶片 + 伺服馬達 + 儲物箱 → 製造機（耗能 20 EU）
- 貨運架構師成就描述已更新

用法範例：
```
機器產出 → [物流鏈] → [貨物輸出節點] → 原版箱子
原版箱子 → [貨物抽取節點] → [物流鏈] → 機器輸入
```

## 2026/03/21

### 🐔 基因雞工程平衡性大修

原先基因雞系統存在嚴重平衡問題：繁殖無冷卻（每秒 1 隻）、激發室無間隔（所有等級每秒 1 個資源）、雞永遠不會消耗。現已全面修正：

#### A. 私人雞舍 — 繁殖冷卻
- 每 **120 秒**繁殖一次（原先每秒一次）→ 約 **30 隻/小時**
- 冷卻中顯示剩餘秒數

#### B. 激發室 — 等級間隔 + 進度累積
不再每 tick 直接產出，改為**累積進度到 100% 才產出**，使用 `productionRate()` 計算：

| 等級 | 基礎週期 | 預估產出/小時 | 資源範例 |
|------|---------|-------------|---------|
| Tier 1 | 30 秒 | 120 | 骨頭、泥土、沙子 |
| Tier 2 | 60 秒 | 60 | 煤炭、線、海綿 |
| Tier 3 | 90 秒 | 40 | 金錠、鐵錠、銅錠 |
| Tier 4 | 150 秒 | 24 | 烈焰棒、乖離珍珠 |
| Tier 5 | 300 秒 | 12 | 鑽石、綠寶石 |
| Tier 6 | 600 秒 | 6 | 獄髓錠 |

- 純度加成：異型合子越少、實際間隔越短（最多加速 10%）
- 激發進度在機器狀態列即時顯示百分比

#### C. 口袋雞壽命機制
口袋雞不再永生，使用一定次數後會**消失**並顯示煙霧 + 死亡音效：

| 用途 | 壽命上限 |
|------|---------|
| 繁殖（私人雞舍） | 每隻親代 30 次 |
| 激發（Tier 1） | 1000 次 |
| 激發（Tier 2） | 500 次 |
| 激發（Tier 3） | 200 次 |
| 激發（Tier 4） | 120 次 |
| 激發（Tier 5） | 60 次 |
| 激發（Tier 6） | 30 次 |

- 口袋雞 lore 上顯示繁殖壽命與激發壽命
- 機器運行時即時顯示剩餘壽命（如 `壽命 28/30`）
- 壽命耗盡的雞會自動從機器中移除

## 2026/03/21

### 🤝 全域信任共享系統（v2 重構）

信任系統從「逐台機器」改為**全域玩家級**，一次設定即可適用於所有現有與未來機器。

| 指令 | 說明 |
|------|------|
| `/tech trust <玩家>` | 將該玩家加入你的全域信任清單 |
| `/tech untrust <玩家>` | 將該玩家從全域信任清單移除 |
| `/tech trustlist` | 查看你的全域信任清單 |

- 被信任的玩家可以打開/操作你的**所有**機器，但**不能拆除**
- 不需要對準機器，隨時隨地都可以執行
- 信任清單重啟後仍保留（`trust.yml`）
- 舊版逐台信任資料會自動遷移至全域

### ⚡ 電網 / 物流支援信任玩家互連

- **修復：**你放發電機、朋友放機台 → 原先吃不到電
- 現在只要任一方 `/tech trust` 對方，兩人的機器即可互連電網與物流
- 雙向信任都能生效（A 信任 B 或 B 信任 A 皆可）
