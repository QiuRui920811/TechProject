# 📋 更新日記

> 此頁記錄每次版本更新的修復與新增內容，方便玩家查閱。

## 2026/03/16

### ⚡ 純電力版機器

新增 **8 台純電力版機器**，將手動機器升級為全自動。放下即自動加工，不需手動右鍵觸發。

- 電動粉碎機、電動壓縮機、電動洗礦機、電動拉線機
- 電動淨化器、電動離心機、電動生質實驗室、電動化學反應器
- 新物品：**自動化模組** — 升級手動機器的關鍵元件
- 耗能約手動版的 2 倍，建議搭配效率升級

### 🌐 介面文字全面中文化

- 將所有 GUI 中殘留的英文替換為中文
- Shift → 蹲下、I/O → 傳輸、XP → 經驗、range → 範圍
- 指令參數提示翻譯：`<amount> [player]` → `<數量> [玩家]`

### 🔧 雜項

- 科技百科手冊隱藏原版進階提示文字

## 2026/03/15

### 🐛 修復

- **瞬移寶珠崩潰**：修復使用瞬移寶珠時伺服器報錯的問題。
- **科技磁石崩潰**：修復磁石吸取物品時可能導致的伺服器錯誤。
- **自動農場種子判定**：修復將種子丟入自動農場時，全部變成種子簇而非作物的問題。現在種子單獨放入產出作物（小麥），種子＋骨粉才會產出種子簇。
- **聊天稱號顯示**：修復稱號在聊天中不顯示的問題。

### ✨ 新增

- **魔法物品系統**：新增 25 種魔法物品，包含魔法結晶（一～三階）、乙太結晶（一～三階）、魔法書封面、魔法玻璃、熔岩水晶、通用護符、亡靈頭骨、來世精華、合成界伏殼、9 種古代符文、奇異乙太黏液等。
- **科技磁石**：新增實用工具「科技磁石」，自動吸取周圍 5 格內的掉落物品。蹲下＋右鍵可切換開關。
