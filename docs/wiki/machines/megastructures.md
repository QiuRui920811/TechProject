# 巨構機器

巨構機器是 四階之上的超終局內容，生產最終級材料，目標是製作歐米茄檔案。

---

## 巨構解鎖鏈

```
恆星鑄造廠 → 星鋼鑄造廠
虛空信標   → 真空工藝艙 → 錨定鍛台
軌道列印機 → 天穹裝配台
時序引擎   → 曲率裝配機 → 事件視界鍛台
熵值艙     → 連續體車床
資料檔案庫 → 遺物掃描儀
                      ↓
              巔峰鍛台 → 歐米茄檔案
```

---
