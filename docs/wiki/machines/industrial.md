# 工業擴展機器

工業擴展新增了 20 台專業機器，涵蓋基礎加工、多方塊結構、發電與冷卻四大類。這些機器構成了從基礎金屬到核能的完整工業鏈。

---

## 概覽

| 階段 | 機器數量 | 說明 |
|---|---|---|
| [基礎工業](#基礎工業) | 7 | 合金、洗礦、離心、壓縮、電解、電爐 |
| [多方塊結構](#多方塊結構) | 4 | 高爐、工業冶煉爐、加壓室、鑄造台 |
| [發電機組](#發電機組) | 4 | 蒸汽、柴油、燃氣渦輪、地熱 |
| [冷卻系統](#冷卻系統) | 4 | 散熱片、水冷、冷卻液循環、低溫塔 |
| 核反應爐 | 1 | 鈾/鈽燃料棒發電 |

## 核反應爐

終極發電設備，使用鈾或鈽燃料棒產生大量電力。

| 屬性 | 值 |
|---|---|
| 階級 | 三～四階 |
| 燃料 | 鈾燃料棒 / 鈽燃料棒 |
| 副產物 | 廢鈾棒 |

| 燃料 | 說明 |
|---|---|
| 鈾燃料棒 | 標準燃料，產出大量電力 + 廢鈾棒 |
| 鈽燃料棒 | 高效燃料，產出超大量電力 + 廢鈾棒 |

> ⚠️ 核反應爐需要搭配冷卻系統運作，冷卻不足會導致停機。廢鈾棒可透過工業離心機回收鈽。

## 工業加工鏈

```
原版礦石 ──[粉碎機]──→ 金屬粉 ──[工業洗礦機]──→ 副產物粉
                                                    ↓
焦爐 ──→ 焦炭 + 雜酚油                    [電爐] → 金屬錠
    ↓           ↓                              ↓
  高爐(鋼)   柴油發電              [合金爐] → 合金錠
    ↓                                  ↓
 工業冶煉爐 → 大馬士革鋼 → 硬化合金 → 強化合金
    ↓
 熔融金屬 ──[鑄造台+模具]──→ 板/齒輪/線/棒/管

電解分離機 → 氫氣/矽晶/硒粉
    ↓
 燃氣渦輪(氫氣發電)

工業離心機 → 濃縮鈾 → 鈾燃料棒 → 核反應爐
    ↓
   鈽 → 鈽燃料棒 → 核反應爐（更高效）
```
