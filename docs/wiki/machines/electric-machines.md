# ⚡ 純電力版機器

純電力版機器是手動機器的全自動升級版本。放下後接上電源即可自動加工，**不需要手動右鍵觸發**。

---

## 概覽

| 特性 | 說明 |
|---|---|
| 操作方式 | 全自動，放入材料即自動加工 |
| 能源消耗 | 約手動版的 **2 倍** |
| 升級支援 | 速度 / 效率 / 堆疊升級 |
| 解鎖條件 | 原手動機器 + 進階製造機 |
| 合成方式 | 在 **進階製造機** 中組裝 |

> 💡 純電力版機器的配方與手動版完全相同，只是不需要手動觸發。適合中後期大規模自動化產線。

---

## 自動化模組

製作任何純電力版機器之前，你需要先合成 **自動化模組**。

| 屬性 | 值 |

| 輸入 | 輸出 |
|---|---|
| 進階電路 + 機械零件 + 速度升級 | 自動化模組 |

## 建議與技巧

- **供電規劃**：純電力版機器耗能約為手動版的 2 倍，請確保有足夠的發電設備
- **效率升級**：安裝效率升級可顯著降低能耗，強烈推薦
- **自動化產線**：搭配物流導管 + 能源節點，建立全自動生產線
- **升級順序**：建議先升級最常用的機器（如粉碎機、壓縮器），再逐步升級其他機器
