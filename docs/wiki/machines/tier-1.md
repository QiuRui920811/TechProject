# 一階 — 基礎機器

一階機器是科技線的起點。它們使用原版材料合成，負責最基本的資源加工。

> 🎨 大部分機器在物品欄中顯示為自訂頭顱材質，讓每台機器都有獨特外觀。下方「方塊」欄位指的是放置後的原版方塊形式。

---

## 碎礦機

| 屬性 | 值 |
|---|---|
| 方塊 | 高爐 |
| 觸發方塊 | 鐵柵欄（放在上方） |
| 解鎖 | 初始可解鎖 |

將礦石研磨為粉末，通常能獲得 **雙倍產出**。碘礦機是**手動結構機**，放置後需在上方放一個**鐵柵欄**，右鍵鐵柵欄即可啟動加工。

### 配方

| 輸入 | 輸出 |
|---|---|
| 鐵礦 | 鐵粉 |
| 銅礦 | 銅粉 |
| 石英 | 矽 |
| 煤炭 | 煤粉 |
| 鐵板 | 回收材料 |

## 壓縮器

| 屬性 | 值 |
|---|---|
| 方塊 | 活塞 |
| 觸發方塊 | 鐵製壓力板（放在上方） |
| 解鎖 | 初始可解鎖 |

將材料壓縮成板材、束材或高密度材料。壓縮器是**手動結構機**，放置後需在上方放一個**鐵製壓力板**（重力壓力板），右鍵壓力板即可啟動加工。

> ⚠️ 壓縮機的「鐵板→鋼板」與「鐵板×2→壓實鐵胚」使用相同材料，開始加工前請用 **配方鎖定**（機器 GUI 中的比較器圖示）選擇想要的配方。詳見[配方鎖定](README.md#配方鎖定)。

### 配方

| 輸入 | 輸出 |
|---|---|
| 鐵錠 | 鐵板 |
| 鐵板 + 煤粉 | 鋼板 |
| 鐵板 ×2 | 壓實鐵胚 |
| 煤粉 + 木棍 | 塑膠 |
| 煤粉 ×2 | 碳纖網 |
| 碳纖網 ×2 | 高密碳塊 |
| 鐵板 + 碳纖網 | 複合面板 |
