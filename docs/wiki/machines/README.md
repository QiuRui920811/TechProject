# ⚙️ 機器百科

TechProject 擁有超過 75 台機器，分為 4 個科技階級和多個類別。

---

## 科技階級

| 階級 | 說明 | 典型能耗 |
|---|---|---|
| **TIER 1** | 基礎工業 — 碎礦、冶煉、壓縮、組裝 | 4-12 EU |
| **TIER 2** | 中階加工 — 精煉、化學、離心、物流 | 9-18 EU |
| **TIER 3** | 高階科技 — 量子、雷射、聚變、星球 | 18-34 EU |
| **TIER 4** | 終局巔峰 — 虛空、暗物質、反物質 | 28-52 EU |
| **巨構** | 超終局 — 星鋼、曲速、巔峰鍛台 | 38-74 EU |

---

## 機器類別

| 類別 | 機器數量 | 說明 |
|---|---|---|
| 核心機器 | ~35 台 | 主要加工與合成 |
| 擴展機器 | ~15 台 | 特殊加工、星球相關 |
| 系統機器 | ~15 台 | 自動化採集與物流 |
| 終局機器 | ~13 台 | TIER4 終局材料生產 |
| 巨構機器 | ~8 台 | 最終級材料生產 |

---

## 如何建造機器

1. 在 **研究台** 中解鎖目標機器
2. 查看該機器的合成配方（科技百科或本 wiki）
3. 在 **工作台** 中按照 3×3 藍圖合成機器方塊
4. 將機器方塊放置在地上
5. 確保旁邊有 **發電機** 或 **能源線纜** 供電
6. **右鍵點擊** 機器打開 GUI，放入材料即可開始加工

---

## 如何供電

- 將發電機放在機器旁邊（相鄰方塊）
- 或使用 **導能線纜** 連接發電機與機器
- 或使用 **能源節點** 建立能源網路
