# ⚙️ 機器百科

科技專案 擁有超過 85 台機器，分為 4 個科技階級和多個類別。

---

## 🎨 頭顱材質

大部分機器在物品欄中會顯示為 **自訂頭顱材質（Player Head）**，讓每台機器都有獨特的外觀圖示，一眼就能辨認。

放置後，使用 **結構預覽** 放置的機器（如碎礦機、壓縮器等多方塊結構）仍會以原版方塊形式呈現在世界中。其餘機器則以頭顱方塊形式放置。

> 💡 已放置的舊機器不受影響，可正常使用。

## 如何建造機器

1. 在 **研究台** 中解鎖目標機器
2. 查看該機器的合成配方（科技百科或本 wiki）
3. 在 **工作台** 中按照 3×3 藍圖合成機器方塊
4. 將機器方塊放置在地上
5. 確保旁邊有 **發電機** 或 **能源線纜** 供電
6. **右鍵點擊** 機器打開介面，放入材料即可開始加工

## 配方鎖定

當一台機器擁有 **多個配方使用相同材料** 時（例如壓縮機的「1×鐵板→鋼板」與「2×鐵板→壓實鐵胚」），機器會按順序匹配第一個符合的配方。這可能導致你放入一組鐵板先被壓成鋼板。

為了解決這個問題，機器 GUI 中提供了 **配方鎖定** 功能：

1. 打開機器介面
2. 如果該機器有多個配方，中間列會出現一個 **比較器圖示**（配方鎖定按鈕）
3. 點擊切換鎖定的配方，每次點擊輪換下一個配方，最後可解鎖回「自動匹配」
4. 鎖定後機器 **只會執行該配方**，忽略其他所有配方

鎖定狀態會持久化儲存，重啟伺服器後保留。

### 受影響的機器

以下機器擁有多個使用相同材料的配方，建議使用配方鎖定：

| 機器 | 衝突配方 |
|---|---|
| 壓縮機 / 電動壓縮機 | 1×鐵板→鋼板 vs 2×鐵板→壓實鐵胚 |
| 自動農場 | 種子→作物 vs 種子+骨粉→種子簇 |
| 回收機 | 木粉→回收材 vs 回收材+木粉→塑膠 |

## 如何供電

- 將發電機放在機器旁邊（相鄰方塊）
- 或使用 **導能線纜** 連接發電機與機器
- 或使用 **能源節點** 建立能源網路
