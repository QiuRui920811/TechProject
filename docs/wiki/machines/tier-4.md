# 四階 — 終局機器

四階機器是科技線的終點，處理虛空、暗物質、反物質等極端材料。

---

## 終局機器解鎖鏈

```
物質編譯器
  ├→ 虛空抽取器 → 暗物質織機
  │                  ↓
  │              重力子穩定器 → 奇點壓縮台
  │                              ↓
  │                          反物質合成器 → 恆星鑄造廠
  │                                          ↓
  │                                      軌道列印機 → 時序引擎
  │                                                      ↓
  ├→ 虛空信標 ←────────────────────────────虛空抽取器
  │
  ├→ 奈米工坊 ←── 化學反應器
  │
  ├→ 熵值艙 ←── 時序引擎
  │
  ├→ 天穹裝配台 ←── 軌道列印機
  │
  └→ 歐米茄製造器 ←── 天穹裝配台
```
