# 四階 — 終局機器

四階機器是科技線的終點，處理虛空、暗物質、反物質等極端材料。

---

## 終局機器解鎖鏈

```
物質編譯器
  ├→ 虛空抽取器 → 暗物質織機
  │                  ↓
  │              重力子穩定器 → 奇點壓縮台
  │                              ↓
  │                          反物質合成器 → 恆星鑄造廠
  │                                          ↓
  │                                      軌道列印機 → 時序引擎
  │                                                      ↓
  ├→ 虛空信標 ←────────────────────────────虛空抽取器
  │
  ├→ 奈米工坊 ←── 化學反應器
  │
  ├→ 熵值艙 ←── 時序引擎
  │
  ├→ 天穹裝配台 ←── 軌道列印機
  │
  └→ 歐米茄製造器 ←── 天穹裝配台
```

## 聚變反應爐

獨立於暗物質/反物質終局線之外的四階能源支線，提供高密度自循環電力。

| 屬性 | 值 |
|---|---|
| 方塊 | 哭泣的黑曜石 |
| 階級 | 四階 |
| 能源緩衝 | 12000 L |
| 解鎖 | 安卓站 → 聚變反應爐研究 |

聚變反應爐**同時**是「機器」和「合成設備」——它本身會發電，但又能用來合成自己的後續零件（聚變網、穩定核心、聚變核心）。

### 合成輸出

| 輸入 | 輸出 | 能量耗用 |
|---|---|---|
| 隔熱外殼 ×2 + 力場板 | 聚變網（fusion_mesh）| 38 EU |
| 磁環 ×2 + 反應爐核心 | 穩定核心（stability_core）| 26 EU |
| 聚變網 ×2 + 穩定核心 | 聚變核心（fusion_core）| 42 EU |

### 聚變零件用途

| 物品 | 中文名 | 用途 |
|---|---|---|
| `fusion_mesh` | 聚變網 | 聚變級耐熱網，後續聚變零件骨架 |
| `stability_core` | 穩定核心 | 高階穩定核心，控制聚變反應 |
| `fusion_core` | 聚變核心 | 聚變能源核心，作為四階配方關鍵材料 |

> 💡 聚變反應爐自循環：先做出聚變網 + 穩定核心，再合成聚變核心。聚變核心供給更上層的終局合成（虛空、暗物質支線）使用。

## 採礦鑽機三型

| 屬性 | 值 |
|---|---|
| 內部 ID | `quarry_drill_mk3` |
| 方塊 | 高爐 |
| 階級 | 四階 |
| 解鎖 | 採礦鑽機二型 + 強化合金錠 |

最高階區塊級採礦設備，挖礦速度與礦脈密度皆遠勝二型。配合**安卓站**和**自動倉儲**可達成 24 小時無人採礦。

## 異界傳送門

| 屬性 | 值 |
|---|---|
| 內部 ID | `otherworld_portal` |
| 分類 | SPECIAL（特殊） |
| 階級 | 四階 |
| 方塊 | 終界傳送門框架 |
| 輸入 | 魔力隕石 |
| 解鎖 | 魔力隕石 + 行星星門 |

**手持魔力隕石右鍵啟動**：消耗 1 顆隕石將玩家投送至異世界。探索結束後由撤離點自動送回。

> ⚠️ 異世界是隨機事件型副本，不固定在某顆星球，內部規則跟主世界不同（環境敵對 / 限時挑戰 / 特殊掉落）。撤離點通常會出現在副本中後期，沒撤回前無法返回主世界。
