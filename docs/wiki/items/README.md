## 物品取得方式

| 方式 | 說明 |
|---|---|
| 淘金盤篩礦 | 手持淘金盤右鍵礫石/靈魂沙（前期核心） |
| 機器配方 | 在對應機器中加工合成 |
| 工作台藍圖 | 在工作台中 3×3 合成 |
| 星球採集 | 在星球世界中獲取 |
| 種植/收穫 | 種植果樹或作物 |
| 怪物掉落 | 星球特殊怪物掉落 |
| 營火烹飪 | 互動式烹飪 |
