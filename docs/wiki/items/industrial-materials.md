### 合金進階路線

```
銅 + 錫 → 青銅（基礎）
鐵 + 焦炭 → 鋼（基礎）
鋼 + 鐵粉 + 碳粉 → 大馬士革鋼（進階）
大馬士革鋼 + 碳化鎢 → 硬化合金（三階）
硬化合金 + 合成鑽石 + 濃縮鈾 → 強化合金（四階 · 終極材料）
```
