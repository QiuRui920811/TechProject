### 合金進階路線

```
銅 + 錫 → 青銅（基礎）
鐵 + 焦炭 → 鋼（基礎）
鋼 + 鐵粉 + 碳粉 → 大馬士革鋼（進階）
大馬士革鋼 + 碳化鎢 → 硬化合金（三階）
硬化合金 + 合成鑽石 + 濃縮鈾 → 強化合金（四階 · 終極材料）
```

## 核電材料

### 燃料循環

```
鈾礦 → [粉碎機] → 鈾礦粉 → [離心機 ×4] → 濃縮鈾
                                        ↓
                          [組裝機] 濃縮鈾 ×4 + 鉛 ×2 + 鋼棒
                                        ↓
                                    鈾燃料棒
                                        ↓
                                 [核反應爐] 一輪 400 tick
                                        ↓
                                    廢鈾棒
                                        ↓
                       [離心機] 鈽 ←──┤
                                        ├→ [電動化學反應器] 廢鈾棒 ×2 + 反應堆冷卻液 → 濃縮鈾（再處理路徑）
                                        ↓
                          [組裝機] 鈽 ×4 + 鉛 ×2 + 鋼棒
                                        ↓
                                  鈽燃料棒（更高效）
```

### 物品列表

| 物品 | 圖示 | 階級 | 說明 | 來源 |
|---|---|---|---|---|
| 鈾礦粉 | 黃綠染料 | 三階 | 鈾礦粉碎產物，可離心提純 | 粉碎機（鈾礦） |
| 濃縮鈾 | 綠色染料 | 三階 | 核反應爐燃料前驅物 | 工業離心機（鈾礦粉 ×4） |
| 鈾燃料棒 | 終界燭 | 三階 | 核反應爐高效燃料 | 組裝機（濃縮鈾 ×4 + 鉛 ×2 + 鋼棒） |
| 廢鈾棒 | 終界燭 | 三階 | 用完的燃料棒，可回收鈽 | 核反應爐每輪輸出 |
| **鈽** | 洋紅染料 | **四階** | 核反應爐副產物，可再利用為燃料 | 工業離心機（廢鈾棒，60 EU） |
| **鈽燃料棒** | 終界燭 | **四階** | 鈽製燃料棒，比鈾更高效 | 組裝機（鈽 ×4 + 鉛 ×2 + 鋼棒，80 EU） |
| 反應爐外殼 | 鐵塊 | 三階 | 鈷鉻合金製耐輻射外殼 | 組裝機（鈷鉻合金錠 ×2 + 鋼板 ×4 + 鉛錠 ×2） |
| 輻射屏蔽板 | 鐵塊 | 三階 | 鉛板製輻射屏蔽 | 加壓室（鉛板 + 鋼板） |
| 合成鑽石 | 鑽石 | 進階 | 高壓壓縮碳粉 | 加壓室（碳粉 ×16） |
| **強化合金錠** | 獄髓錠 | **四階** | 最高等級的合金，終局製造的骨架 | 工業冶煉廠（硬化合金錠 + 合成鑽石 + 濃縮鈾，150 EU） |

> 💡 **鈽 vs 鈾**：鈽燃料棒運轉效率比鈾燃料棒高，但鈽只能從廢鈾棒離心取得——所以**先燒鈾、再煉鈽**是固定流程。
>
> 💡 **強化合金錠**是四階機器的骨架材料（終局機具用），需要硬化合金 + 合成鑽石 + 濃縮鈾三料，是核電線跟工業線的交集成果。
