# 配方系統

TechProject 包含三種配方體系：工作台合成、機器加工、互動烹飪。

---

## 配方類型

| 類型 | 說明 | 解鎖方式 |
|---|---|---|
| [工作台配方](workbench.md) | 3×3 工作台合成，用於建造機器 | 研究台解鎖藍圖 |
| [機器配方](machine-recipes.md) | 放入機器自動加工 | 解鎖對應機器 |
| [烹飪配方](cooking.md) | 營火 / 煙燻爐互動烹飪 | 手持食材右鍵點擊 |

---

## 配方取得方式

1. **研究台解鎖**：大部分配方需在研究台花費研究 XP 解鎖
2. **科技百科查詢**：使用 `/tech book` 開啟導覽書，內含所有配方教學
3. **成就獎勵**：部分進階合成方法會在完成特定成就後提示
