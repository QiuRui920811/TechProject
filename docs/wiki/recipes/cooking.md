# 互動烹飪

手持食材在營火、煙燻爐或高爐上按右鍵即可開始烹飪。

---

## 烹飪方式

1. 放置營火或煙燻爐
2. 手持對應食材
3. 對準方塊按右鍵
4. 等待烹飪完成後，物品會自動掉落
