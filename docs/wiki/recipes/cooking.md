# 互動烹飪

手持食材在營火、煙燻爐或高爐上按右鍵即可開始烹飪。

---

## 烹飪方式

1. 放置營火或煙燻爐
2. 手持對應食材
3. 對準方塊按右鍵
4. 烹飪過程中留意互動提示（翻面 / 攪拌 / 調味），及時右鍵可加速烹調
5. 等待烹飪完成後，物品會自動掉落

---

## 互動烹飪系統

烹飪過程中會依進度觸發 **3 次互動提示**：

| 進度 | 互動動作 | 提示 |
|---|---|---|
| 30% | 翻面 | 🔔 點擊翻面！ |
| 55% | 攪拌 | 🔔 點擊攪拌！ |
| 80% | 調味 | 🔔 點擊調味！ |

- 看到提示時對準烹飪台 **右鍵** 即可觸發互動
- 每次成功互動會 **加速烹調**，並播放翻轉動畫與粒子特效
- 有 3 秒的互動窗口，超時不影響烹調但不會獲得加速
- 三次互動都成功可觸發「🌟 完美烹調」特效

### 視覺動畫

- 食物會懸浮在烹飪台上方旋轉，旋轉速度隨進度加快
- BossBar 即時顯示進度條與烹調階段
- 五個烹調階段各有不同粒子效果（煙霧 → 蒸氣 → 金色光點 → 星光）
- 互動提示期間食物會脈動閃爍並圍繞光環
